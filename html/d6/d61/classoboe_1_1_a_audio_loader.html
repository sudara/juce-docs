Detailed Description

The AAudio API was not available in early versions of Android. To avoid linker errors, we dynamically link with the functions by name using dlsym(). On older versions this linkage will safely fail.

◆ open()

int oboe::AAudioLoader::open ( )

Open the AAudio shared library and load the function pointers. This can be called multiple times. It should only be called from one thread.

The destructor will clean up after the open.

Returns
0 if successful or negative error.
