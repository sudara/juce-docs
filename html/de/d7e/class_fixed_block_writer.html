Detailed Description

This can be used to convert a push data flow from variable sized buffers to fixed sized buffers. An example would be an audio input callback.

Write from a variable sized block.

Note that if the fixed-sized blocks must be aligned, then the variable-sized blocks must have the same alignment. For example, if the fixed-size blocks must be a multiple of 8, then the variable-sized blocks must also be a multiple of 8.
