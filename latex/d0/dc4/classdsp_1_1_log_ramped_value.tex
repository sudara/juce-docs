\subsubsection*{template$<$typename \mbox{\hyperlink{class_smoothed_value_base_a331eba8c0333698d8bb470b1311d49e4}{Float\+Type}}$>$\newline
class dsp\+::\+Log\+Ramped\+Value$<$ Float\+Type $>$}
Utility class for logarithmically smoothed linear values.

Logarithmically smoothed values can be more relevant than linear ones for specific cases such as algorithm change smoothing, using two of them in opposite directions.

The gradient of the logarithmic/exponential slope can be configured by calling \mbox{\hyperlink{classdsp_1_1_log_ramped_value_a952fc8a88b07a6e46dbb24d676a697bd}{Log\+Ramped\+Value\+::set\+Log\+Parameters}}.
