\doxysubsection{Detailed Description}
An implementation of \mbox{\hyperlink{class_tooltip_client}{Tooltip\+Client}} that stores the tooltip string and a method for changing it.

This makes it easy to add a tooltip to a custom component, by simply adding this as a base class and calling \mbox{\hyperlink{class_settable_tooltip_client_aa374ef7cc0eea80f3922f5d584da79e9}{set\+Tooltip()}}.

Many of the JUCE widgets already use this as a base class to implement their tooltips.
