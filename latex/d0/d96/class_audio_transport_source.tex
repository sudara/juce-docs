\doxysubsection{Detailed Description}
An \mbox{\hyperlink{class_audio_source}{Audio\+Source}} that takes a \mbox{\hyperlink{class_positionable_audio_source}{Positionable\+Audio\+Source}} and allows it to be played, stopped, started, etc.

This can also be told use a buffer and background thread to read ahead, and if can correct for different sample-\/rates.

You may want to use one of these along with an \mbox{\hyperlink{class_audio_source_player}{Audio\+Source\+Player}} and \mbox{\hyperlink{class_audio_i_o_device}{Audio\+IODevice}} to control playback of an audio file.

The source passed in will not be deleted by this object, so must be managed by the caller.
