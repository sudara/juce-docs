\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Value\+Type$>$\newline
class Border\+Size$<$ Value\+Type $>$}
Specifies a set of gaps to be left around the sides of a rectangle.

This is basically the size of the spaces at the top, bottom, left and right of a rectangle. It\textquotesingle{}s used by various component classes to specify borders.
