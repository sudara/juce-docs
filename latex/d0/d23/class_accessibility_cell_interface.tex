\doxysubsection{Detailed Description}
An abstract interface which represents a UI element that supports a cell interface.

This typically represents a single cell inside of a UI element which implements an \mbox{\hyperlink{class_accessibility_table_interface}{Accessibility\+Table\+Interface}}.
