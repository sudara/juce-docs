A note on use of preprocessor defines\+:

This is one of the few times when it\textquotesingle{}s suitable to use preprocessor defines rather than constexpr Why? Because C++11 requires a lot of boilerplate code to convert integers into compile-\/time string literals. The preprocessor, despite it\textquotesingle{}s lack of type checking, is more suited to the task
