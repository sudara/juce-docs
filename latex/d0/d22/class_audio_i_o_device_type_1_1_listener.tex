\doxysubsection{Detailed Description}
A class for receiving events when audio devices are inserted or removed.

You can register an \mbox{\hyperlink{class_audio_i_o_device_type_1_1_listener}{Audio\+IODevice\+Type\+::\+Listener}} with an$\sim$\+Audio\+IODevice\+Type object using the \mbox{\hyperlink{class_audio_i_o_device_type_aba66d5fc04bb8162ad082bbc5a46df37}{Audio\+IODevice\+Type\+::add\+Listener()}} method, and it will be called when devices of that type are added or removed.
