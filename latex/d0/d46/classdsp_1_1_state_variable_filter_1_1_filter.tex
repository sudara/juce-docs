\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Sample\+Type$>$\newline
class dsp\+::\+State\+Variable\+Filter\+::\+Filter$<$ Sample\+Type $>$}
An \mbox{\hyperlink{namespacedsp_1_1_i_i_r}{IIR}} filter that can perform low, band and high-\/pass filtering on an audio signal, with 12 dB of attenuation per octave, using a TPT structure, designed for fast modulation (see Vadim Zavalishin\textquotesingle{}s documentation about TPT structures for more information). Its behaviour is based on the analog state variable filter circuit.

Note\+: The bandpass here is not the one in the RBJ Cook\+Book as its gain can be higher than 0 dB. For the classic 0 dB bandpass, we need to multiply the result by R2.

Note 2\+: Using this class prevents some loud audio artefacts commonly encountered when changing the cutoff frequency using other filter simulation structures and \mbox{\hyperlink{namespacedsp_1_1_i_i_r}{IIR}} filter classes. However, this class may still require additional smoothing for cutoff frequency changes.

see \mbox{\hyperlink{class_i_i_r_filter}{IIRFilter}}, \mbox{\hyperlink{class_smoothed_value}{Smoothed\+Value}}
