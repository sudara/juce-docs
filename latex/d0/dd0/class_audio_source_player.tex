\doxysubsection{Detailed Description}
Wrapper class to continuously stream audio from an audio source to an \mbox{\hyperlink{class_audio_i_o_device}{Audio\+IODevice}}.

This object acts as an \mbox{\hyperlink{class_audio_i_o_device_callback}{Audio\+IODevice\+Callback}}, so can be attached to an output device, and will stream audio from an \mbox{\hyperlink{class_audio_source}{Audio\+Source}}.

Changes the current audio source to play from.

If the source passed in is already being used, this method will do nothing. If the source is not null, its \mbox{\hyperlink{class_audio_source_player_a0941ec351494d77bce92df935c98b777}{prepare\+To\+Play()}} method will be called before it starts being used for playback.

If there\textquotesingle{}s another source currently playing, its release\+Resources() method will be called after it has been swapped for the new one.
