\doxysubsection{Detailed Description}
Represents a mouse cursor image.

This object can either be used to represent one of the standard mouse cursor shapes, or a custom one generated from an image.

{\footnotesize\ttfamily static \mbox{\hyperlink{png_8h_aa8c59027f9ab2769342f248709d68d17}{void}} Mouse\+Cursor\+::show\+Wait\+Cursor (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [static]}}

Makes the system show its default \textquotesingle{}busy\textquotesingle{} cursor.

This will turn the system cursor to an hourglass or spinning beachball until the next time the mouse is moved, or \mbox{\hyperlink{class_mouse_cursor_a4671d39f6abcc1ca73fd8d95cf1e4355}{hide\+Wait\+Cursor()}} is called.

This is handy if the message loop is about to block for a couple of seconds while busy and you want to give the user feedback about this.
