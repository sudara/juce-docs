\doxysubsection{Detailed Description}
Represents a \mbox{\hyperlink{class_flex_box}{Flex\+Box}} container, which contains and manages the layout of a set of \mbox{\hyperlink{class_flex_item}{Flex\+Item}} objects.

To use this class, set its parameters appropriately (you can search online for more help on exactly how the \mbox{\hyperlink{class_flex_box}{Flex\+Box}} protocol works!), then add your sub-\/items to the items array, and call \mbox{\hyperlink{class_flex_box_a191d589377bc55d4fb9a545bad12e2c9}{perform\+Layout()}} in the resized() function of your \mbox{\hyperlink{class_component}{Component}}.
