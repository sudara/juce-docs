\doxysubsection{Detailed Description}
Uses RAII to save and restore the state of a graphics context. On construction, this calls \mbox{\hyperlink{class_graphics_ab4df35938684890f7adac0439e900ca8}{Graphics\+::save\+State()}}, and on destruction it calls \mbox{\hyperlink{class_graphics_ac724e99c9a7bcd8b2987a484e269a368}{Graphics\+::restore\+State()}} on the \mbox{\hyperlink{class_graphics}{Graphics}} object that you supply.
