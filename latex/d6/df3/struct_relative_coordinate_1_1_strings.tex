\doxysubsection{Detailed Description}
A set of static strings that are commonly used by the \mbox{\hyperlink{class_relative_coordinate}{Relative\+Coordinate}} class.

As well as avoiding using string literals in your code, using these preset values has the advantage that all instances of the same string will share the same, reference-\/counted \mbox{\hyperlink{class_string}{String}} object, so if you have thousands of points which all refer to the same anchor points, this can save a significant amount of memory allocation.
