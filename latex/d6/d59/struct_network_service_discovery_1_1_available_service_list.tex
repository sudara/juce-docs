\doxysubsection{Detailed Description}
Watches the network for broadcasts from \mbox{\hyperlink{struct_network_service_discovery_1_1_advertiser}{Advertiser}} objects, and keeps a list of all the currently active instances.

Just create an instance of \mbox{\hyperlink{struct_network_service_discovery_1_1_available_service_list}{Available\+Service\+List}} and it will start listening -\/ you can register a callback with its on\+Change member to find out when services appear/disappear, and you can call \mbox{\hyperlink{struct_network_service_discovery_1_1_available_service_list_a6c281f52de09c0d7dabfc8d20e8b130b}{get\+Services()}} to find out the current list.
