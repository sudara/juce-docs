\doxysubsection{Detailed Description}
A class that imposes restrictions on a \mbox{\hyperlink{class_component}{Component}}\textquotesingle{}s size or position.

This is used by classes such as \mbox{\hyperlink{class_resizable_corner_component}{Resizable\+Corner\+Component}}, \mbox{\hyperlink{class_resizable_border_component}{Resizable\+Border\+Component}} and \mbox{\hyperlink{class_resizable_window}{Resizable\+Window}}.

The base class can impose some basic size and position limits, but you can also subclass this for custom uses.

Sets the amount by which the component is allowed to go off-\/screen.

The values indicate how many pixels must remain on-\/screen when dragged off one of its parent\textquotesingle{}s edges, so e.\+g. if minimum\+When\+Off\+The\+Top is set to 10, then when the component goes off the top of the screen, its y-\/position will be clipped so that there are always at least 10 pixels on-\/screen. In other words, the lowest y-\/position it can take would be (10 -\/ the component\textquotesingle{}s height).

If you pass 0 or less for one of these amounts, the component is allowed to move beyond that edge completely, with no restrictions at all.

If you pass a very large number (i.\+e. larger that the dimensions of the component itself), then the component won\textquotesingle{}t be allowed to overlap that edge at all. So e.\+g. setting minimum\+When\+Off\+The\+Left to 0xffffff will mean that the component will bump into the left side of the screen and go no further. \mbox{\Hypertarget{class_component_bounds_constrainer_aea5a0319cabade466756b4f031b9329d}\label{class_component_bounds_constrainer_aea5a0319cabade466756b4f031b9329d}}
