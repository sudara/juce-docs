\doxysubsection{Detailed Description}
A stream which uses zlib to compress the data written into it.

Important note\+: When you call \mbox{\hyperlink{class_g_z_i_p_compressor_output_stream_a4b2e3135b2b9707edaf8fac8d74d557a}{flush()}} on a \mbox{\hyperlink{class_g_z_i_p_compressor_output_stream}{GZIPCompressor\+Output\+Stream}}, the gzip data is closed -\/ this means that no more data can be written to it, and any subsequent attempts to call \mbox{\hyperlink{class_g_z_i_p_compressor_output_stream_a65127167237b8f998cbad09622edec3a}{write()}} will cause an assertion.
