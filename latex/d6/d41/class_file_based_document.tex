\doxysubsection{Detailed Description}
A class to take care of the logic involved with the loading/saving of some kind of document.

There\textquotesingle{}s quite a lot of tedious logic involved in writing all the load/save/save-\/as functions you need for documents that get saved to a file, so this class attempts to abstract most of the boring stuff.

Your subclass should just implement all the pure virtual methods, and you can then use the higher-\/level public methods to do the load/save dialogs, to warn the user about overwriting files, etc.

The document object keeps track of whether it has changed since it was last saved or loaded, so when you change something, call its \mbox{\hyperlink{class_file_based_document_a3a058b73776404d76b828205ca058e42}{changed()}} method. This will set a flag so it knows it needs saving, and will also broadcast a change message using the \mbox{\hyperlink{class_change_broadcaster}{Change\+Broadcaster}} base class.

Called to indicate that the document has changed and needs saving.

This is used for dialog boxes to make them open at the last folder you were using.

If you\textquotesingle{}ve got a document open and want to close it (e.\+g. to quit the app), this is the method to call.

If the document doesn\textquotesingle{}t need saving the callback will be called with the value saved\+Ok so you can go ahead and delete the document.

If it does need saving it\textquotesingle{}ll prompt the user, and if they say \char`\"{}discard changes\char`\"{} the callback will be called with saved\+Ok, so again, you can safely delete the document.
