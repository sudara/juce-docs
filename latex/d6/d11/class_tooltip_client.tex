\doxysubsection{Detailed Description}
Components that want to use pop-\/up tooltips should implement this interface.

A \mbox{\hyperlink{class_tooltip_window}{Tooltip\+Window}} will wait for the mouse to hover over a component that implements the \mbox{\hyperlink{class_tooltip_client}{Tooltip\+Client}} interface, and when it finds one, it will display the tooltip returned by its \mbox{\hyperlink{class_tooltip_client_a233635a4125d43bd54e9660c4ba40a90}{get\+Tooltip()}} method.
