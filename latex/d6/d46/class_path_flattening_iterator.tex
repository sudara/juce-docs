\doxysubsection{Detailed Description}
Flattens a \mbox{\hyperlink{class_path}{Path}} object into a series of straight-\/line sections.

Use one of these to iterate through a \mbox{\hyperlink{class_path}{Path}} object, and it will convert all the curves into line sections so it\textquotesingle{}s easy to render or perform geometric operations on.
