\doxysubsection{Detailed Description}
A class encapsulating a group of Audio\+Processor\+Parameters and nested Audio\+Processor\+Parameter\+Groups.

This class is predominantly write-\/only; there are methods for adding group members but none for removing them. Ultimately you will probably want to add a fully constructed group to an \mbox{\hyperlink{class_audio_processor}{Audio\+Processor}}.
