\doxysubsection{Detailed Description}
Wraps another input stream, and reads from it using an intermediate buffer

If you\textquotesingle{}re using an input stream such as a file input stream, and making lots of small read accesses to it, it\textquotesingle{}s probably sensible to wrap it in one of these, so that the source stream gets accessed in larger chunk sizes, meaning less work for the underlying stream.

Tries to move the current read position of the stream.

The position is an absolute number of bytes from the stream\textquotesingle{}s start.
