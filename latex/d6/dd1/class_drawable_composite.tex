\doxysubsection{Detailed Description}
A drawable object which acts as a container for a set of other Drawables.

Note that although this is a \mbox{\hyperlink{class_component}{Component}}, it takes ownership of its child components and will delete them, so that you can use it as a self-\/contained graphic object. The intention is that you should not add your own components to it, only add other \mbox{\hyperlink{class_drawable}{Drawable}} objects.
