\doxysubsection{Detailed Description}
An RAII class for sending slider listener drag messages.

This is useful if you are programmatically updating the slider\textquotesingle{}s value and want to imitate a mouse event, for example in a custom \mbox{\hyperlink{class_accessibility_handler}{Accessibility\+Handler}}.
