\doxysubsection{Detailed Description}
An OSC argument.

An OSC argument is a value of one of the following types\+: int32, float32, string, or blob (raw binary data).

\mbox{\hyperlink{class_o_s_c_message}{OSCMessage}} objects are essentially arrays of \mbox{\hyperlink{class_o_s_c_argument}{OSCArgument}} objects.
