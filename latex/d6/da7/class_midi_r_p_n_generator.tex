\doxysubsection{Detailed Description}
Generates an appropriate sequence of MIDI CC messages to represent an RPN or NRPN message.

This sequence (as a \mbox{\hyperlink{class_midi_buffer}{Midi\+Buffer}}) can then be directly sent to a \mbox{\hyperlink{class_midi_output}{Midi\+Output}}.
