Converts MIDI 1 Universal MIDI \mbox{\hyperlink{classuniversal__midi__packets_1_1_packets}{Packets}} to corresponding MIDI 2 packets, calling {\ttfamily callback} with each converted packet.

In some cases (such as RPN/\+NRPN messages) multiple MIDI 1 packets will convert to a single MIDI 2 packet. In these cases, the translator will accumulate the full message internally, and send a single callback with the completed message, once all the individual MIDI 1 packets have been processed. \mbox{\Hypertarget{classuniversal__midi__packets_1_1_midi1_to_midi2_default_translator_a016f92f0727315b0e9ae88813ff0382a}\label{classuniversal__midi__packets_1_1_midi1_to_midi2_default_translator_a016f92f0727315b0e9ae88813ff0382a}}
