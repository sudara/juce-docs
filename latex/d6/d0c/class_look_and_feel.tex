\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_look_and_feel}{Look\+And\+Feel}} objects define the appearance of all the JUCE widgets, and subclasses can be used to apply different \textquotesingle{}skins\textquotesingle{} to the application.

This class is an abstract base-\/class -\/ for actual look-\/and-\/feels that you can instantiate, see \mbox{\hyperlink{class_look_and_feel___v1}{Look\+And\+Feel\+\_\+\+V1}}, \mbox{\hyperlink{class_look_and_feel___v2}{Look\+And\+Feel\+\_\+\+V2}} and \mbox{\hyperlink{class_look_and_feel___v3}{Look\+And\+Feel\+\_\+\+V3}}.

{\footnotesize\ttfamily \mbox{\hyperlink{class_colour}{Colour}} Look\+And\+Feel\+::find\+Colour (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{group__juce__audio__plugin__client-_unity_ga966b05a42d98c7296097e073c627dcd0}{int}}}]{colour\+Id }\end{DoxyParamCaption}) const\hspace{0.3cm}{\ttfamily [noexcept]}}

Looks for a colour that has been registered with the given colour ID number.

If a colour has been set for this ID number using \mbox{\hyperlink{class_look_and_feel_a167a1e914771f32e433a0d45aaba45e3}{set\+Colour()}}, then it is returned. If none has been set, it will just return \mbox{\hyperlink{namespace_colours_ab5f46ac613c2b717ac885b7cf995ffc6}{Colours\+::black}}.

The colour IDs for various purposes are stored as enums in the components that they are relevant to -\/ for an example, see \mbox{\hyperlink{class_slider_a1012002c53381ccc7c1fe7e604a75f44}{Slider\+::\+Colour\+Ids}}, \mbox{\hyperlink{class_label_a41756012394513222e1323bb432fa701}{Label\+::\+Colour\+Ids}}, \mbox{\hyperlink{class_text_editor_aa805220923c93cd808a92fff0e3fb71d}{Text\+Editor\+::\+Colour\+Ids}}, \mbox{\hyperlink{class_tree_view_abeebe6dbdc6ae466495aabc64eb51d6a}{Tree\+View\+::\+Colour\+Ids}}, etc.

If you\textquotesingle{}re looking up a colour for use in drawing a component, it\textquotesingle{}s usually best not to call this directly, but to use the \mbox{\hyperlink{class_component_a743504b7a85b697404bf9aab17324265}{Component\+::find\+Colour()}} method instead. That will first check whether a suitable colour has been registered directly with the component, and will fall-\/back on calling the component\textquotesingle{}s \mbox{\hyperlink{class_look_and_feel}{Look\+And\+Feel}}\textquotesingle{}s \mbox{\hyperlink{class_look_and_feel_aa7f9faa480c9ec9d61b66672425f380b}{find\+Colour()}} method if none is found.
