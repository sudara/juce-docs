\doxysubsection{Detailed Description}
A utility object that helps you initialise and shutdown JUCE correctly using an RAII pattern.

When the first instance of this class is created, it calls \mbox{\hyperlink{group__juce__events-messages_gaa648b242fe6ecc639ac53ec9c969822f}{initialise\+Juce\+\_\+\+GUI()}}, and when the last instance is deleted, it calls \mbox{\hyperlink{group__juce__events-messages_ga1c949ca46c8ef9bcefbaf9256ab6ff80}{shutdown\+Juce\+\_\+\+GUI()}}, so that you can easily be sure that as long as at least one instance of the class exists, the library will be initialised.

This class is particularly handy to use at the beginning of a console app\textquotesingle{}s main() function, because it\textquotesingle{}ll take care of shutting down whenever you return from the main() call.

Be careful with your threading though -\/ to be safe, you should always make sure that these objects are created and deleted on the message thread.
