\subsubsection*{template$<$typename Callback\+Type = Message\+Loop\+Callback$>$\newline
class OSCReceiver\+::\+Listener$<$ Callback\+Type $>$}
A class for receiving OSC data from an \mbox{\hyperlink{class_o_s_c_receiver}{OSCReceiver}}.

The template argument Callback\+Type determines how the callback will be called and has to be either \mbox{\hyperlink{struct_o_s_c_receiver_1_1_message_loop_callback}{Message\+Loop\+Callback}} or \mbox{\hyperlink{struct_o_s_c_receiver_1_1_realtime_callback}{Realtime\+Callback}}. If not specified, \mbox{\hyperlink{struct_o_s_c_receiver_1_1_message_loop_callback}{Message\+Loop\+Callback}} will be used by default.
