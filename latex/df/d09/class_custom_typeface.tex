\doxysubsection{Detailed Description}
A typeface that can be populated with custom glyphs.

You can create a \mbox{\hyperlink{class_custom_typeface}{Custom\+Typeface}} if you need one that contains your own glyphs, or if you need to load a typeface from a Juce-\/formatted binary stream.

If you want to create a copy of a native face, you can use \mbox{\hyperlink{class_custom_typeface_a94de59c3b010a1f68ce3681174a6a173}{add\+Glyphs\+From\+Other\+Typeface()}} to copy glyphs into this face.

NOTE! For most people this class is almost certainly NOT the right tool to use! If what you want to do is to embed a font into your exe, then your best plan is probably to embed your TTF/\+OTF font file into your binary using the Projucer, and then call \mbox{\hyperlink{class_typeface_a2ce51562cc51437352c4d47777185f35}{Typeface\+::create\+System\+Typeface\+For()}} to load it from memory.
