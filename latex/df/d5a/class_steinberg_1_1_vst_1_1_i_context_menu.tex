\doxysubsection{Detailed Description}
Context Menu interface\+: \mbox{\hyperlink{class_steinberg_1_1_vst_1_1_i_context_menu}{Vst\+::\+IContext\+Menu}}


\begin{DoxyItemize}
\item \mbox{[}host imp\mbox{]}
\item \mbox{[}create with \mbox{\hyperlink{class_steinberg_1_1_vst_1_1_i_component_handler3_a9a40caa40f849abbc216a2ea58ee2768}{IComponent\+Handler3\+::create\+Context\+Menu}}(..)\mbox{]}
\item \mbox{[}released\+: 3.\+5.\+0\mbox{]}
\item \mbox{[}optional\mbox{]}
\end{DoxyItemize}

A context menu is composed of Item (entry). A Item is defined by a name, a tag, a flag and a associated target (called when this item will be selected/executed). With \mbox{\hyperlink{class_steinberg_1_1_vst_1_1_i_context_menu}{IContext\+Menu}} the plug-\/in can retrieve a Item, add a Item, remove a Item and pop-\/up the menu.
