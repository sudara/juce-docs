\doxysubsection{Detailed Description}
Used to convert strings to localised foreign-\/language versions.

This is basically a look-\/up table of strings and their translated equivalents. It can be loaded from a text file, so that you can supply a set of localised versions of strings that you use in your app.

To use it in your code, simply call the \mbox{\hyperlink{class_localised_strings_a5545fd7648fcd404913fbba46c1eee4b}{translate()}} method on each string that might have foreign versions, and if none is found, the method will just return the original string.

The translation file should start with some lines specifying a description of the language it contains, and also a list of ISO country codes where it might be appropriate to use the file. After that, each line of the file should contain a pair of quoted strings with an \textquotesingle{}=\textquotesingle{} sign.

E.\+g. for a french translation, the file might be\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{language: French}
\DoxyCodeLine{countries: fr be mc ch lu}
\DoxyCodeLine{}
\DoxyCodeLine{\textcolor{stringliteral}{"{}hello"{}} = \textcolor{stringliteral}{"{}bonjour"{}}}
\DoxyCodeLine{\textcolor{stringliteral}{"{}goodbye"{}} = \textcolor{stringliteral}{"{}au revoir"{}}}

\end{DoxyCode}


If the strings need to contain a quote character, they can use \textquotesingle{}"{}\textquotesingle{} instead, and if the first non-\/whitespace character on a line isn\textquotesingle{}t a quote, then it\textquotesingle{}s ignored, (you can use this to add comments).

Note that this is a singleton class, so don\textquotesingle{}t create or destroy the object directly. There\textquotesingle{}s also a \mbox{\hyperlink{group__juce__core-text_gaf3c28ad27b0d8fa2f615888701e32b07}{TRANS(text)}} macro defined to make it easy to use the this.

E.\+g. 
\begin{DoxyCode}{0}
\DoxyCodeLine{printSomething (\mbox{\hyperlink{group__juce__core-text_gaf3c28ad27b0d8fa2f615888701e32b07}{TRANS}}(\textcolor{stringliteral}{"{}hello"{}}));}

\end{DoxyCode}


This macro is used in the JUCE classes themselves, so your application has a chance to intercept and translate any internal JUCE text strings that might be shown. (You can easily get a list of all the messages by searching for the \mbox{\hyperlink{group__juce__core-text_gaf3c28ad27b0d8fa2f615888701e32b07}{TRANS()}} macro in the JUCE source code).
