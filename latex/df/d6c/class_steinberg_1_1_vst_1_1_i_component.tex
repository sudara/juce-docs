\mbox{\hyperlink{class_steinberg_1_1_vst_1_1_component}{Component}} base interface\+: \mbox{\hyperlink{class_steinberg_1_1_vst_1_1_i_component}{Vst\+::\+IComponent}}

\item \mbox{[}mandatory\mbox{]}

This is the basic interface for a VST component and must always be supported. It contains the common parts of any kind of processing class. The parts that are specific to a media type are defined in a separate interface. An implementation component must provide both the specific interface and \mbox{\hyperlink{class_steinberg_1_1_vst_1_1_i_component}{IComponent}}. \begin{DoxySeeAlso}{See also}
