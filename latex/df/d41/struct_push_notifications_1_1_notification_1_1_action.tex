\doxysubsection{Detailed Description}
Represents an action on a notification that can be presented as a button or a text input. On Android, each notification has its action specified explicitly, on i\+OS you configure an allowed set of actions on startup and pack them into categories (see \mbox{\hyperlink{struct_push_notifications_1_1_settings}{Settings}}).
