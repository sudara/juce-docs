{\footnotesize\ttfamily \mbox{\hyperlink{class_relative_coordinate}{Relative\+Coordinate}} Marker\+List\+::\+Marker\+::position}

The marker\textquotesingle{}s position.

The expression used to define the coordinate may use the names of other markers, so that markers can be linked in arbitrary ways, but be careful not to create recursive loops of markers whose positions are based on each other! It can also refer to \char`\"{}parent.\+right\char`\"{} and \char`\"{}parent.\+bottom\char`\"{} so that you can set markers which are relative to the size of the component that contains them.

To resolve the coordinate, you can use the \mbox{\hyperlink{class_marker_list_aee0226fdfaf16ec3a2e9e2ca7b433e17}{Marker\+List\+::get\+Marker\+Position()}} method.
