{\footnotesize\ttfamily \mbox{\hyperlink{class_random}{Random}} Unit\+Test\+::get\+Random (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption}) const}

Returns a shared RNG that all unit tests should use. If a test needs random numbers, it\textquotesingle{}s important that when an error is found, the exact circumstances can be re-\/created in order to re-\/test the problem, by repeating the test with the same random seed value. To make this possible, the \mbox{\hyperlink{class_unit_test_runner}{Unit\+Test\+Runner}} class creates a master seed value for the run, writes this number to the log, and then this method returns a \mbox{\hyperlink{class_random}{Random}} object based on that seed. All tests should only use this method to create any \mbox{\hyperlink{class_random}{Random}} objects that they need.

Note that this method will return an identical object each time it\textquotesingle{}s called for a given run, so if you need several different \mbox{\hyperlink{class_random}{Random}} objects, the best way to do that is to call \mbox{\hyperlink{class_random_af8a7c8f1f8aa96b761c441f34382cd4a}{Random\+::combine\+Seed()}} on the result to permute it with a constant value. \mbox{\Hypertarget{class_unit_test_ae244a00deb268069bd59d93169d46440}\label{class_unit_test_ae244a00deb268069bd59d93169d46440}}
