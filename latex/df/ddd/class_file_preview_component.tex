\doxysubsection{Detailed Description}
Base class for components that live inside a file chooser dialog box and show previews of the files that get selected.

One of these allows special extra information to be displayed for files in a dialog box as the user selects them. Each time the current file or directory is changed, the \mbox{\hyperlink{class_file_preview_component_aa89507ce51322a1462bb12d1fd07715a}{selected\+File\+Changed()}} method will be called to allow it to update itself appropriately.
