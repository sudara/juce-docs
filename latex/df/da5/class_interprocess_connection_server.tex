\doxysubsection{Detailed Description}
An object that waits for client sockets to connect to a port on this host, and creates \mbox{\hyperlink{class_interprocess_connection}{Interprocess\+Connection}} objects for each one.

To use this, create a class derived from it which implements the \mbox{\hyperlink{class_interprocess_connection_server_a94b5fa117974fda8748c54f2e526de88}{create\+Connection\+Object()}} method, so that it creates suitable connection objects for each client that tries to connect.

Starts an internal thread which listens on the given port number.

While this is running, if another process tries to connect with the \mbox{\hyperlink{class_interprocess_connection_a007450a6ca6dc60bb48d384479d9a49d}{Interprocess\+Connection\+::connect\+To\+Socket()}} method, this object will call \mbox{\hyperlink{class_interprocess_connection_server_a94b5fa117974fda8748c54f2e526de88}{create\+Connection\+Object()}} to create a connection to that client.

Use \mbox{\hyperlink{class_interprocess_connection_server_ac8a4c37b9e77848e48b499f441749324}{stop()}} to stop the thread running.

Creates a suitable connection object for a client process that wants to connect to this one.

This will be called by the listener thread when a client process tries to connect, and must return a new \mbox{\hyperlink{class_interprocess_connection}{Interprocess\+Connection}} object that will then run as this end of the connection.
