\doxysubsection{Detailed Description}
A set of general-\/purpose application command IDs.

Because these commands are likely to be used in most apps, they\textquotesingle{}re defined here to help different apps to use the same numeric values for them.

Of course you don\textquotesingle{}t have to use these, but some of them are used internally by JUCE -\/ e.\+g. the quit ID is recognised as a command by the \mbox{\hyperlink{class_j_u_c_e_application}{JUCEApplication}} class.
