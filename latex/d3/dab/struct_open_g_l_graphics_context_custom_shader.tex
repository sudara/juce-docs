\doxysubsection{Detailed Description}
Used to create custom shaders for use with an open\+GL 2D rendering context.

Given a GL-\/based rendering context, you can write a fragment shader that applies some kind of per-\/pixel effect.

Creates a custom shader.

The shader code will not be compiled until actually needed, so it\textquotesingle{}s OK to call this constructor when no GL context is active.

The code should be a normal fragment shader. As well as the usual GLSL variables, there is also an automatically declared varying vec2 called \char`\"{}pixel\+Pos\char`\"{}, which indicates the pixel position within the graphics context of the pixel being drawn. There is also a varying value \char`\"{}pixel\+Alpha\char`\"{}, which indicates the alpha by which the pixel should be multiplied, so that the edges of any clip-\/region masks are anti-\/aliased correctly. \mbox{\Hypertarget{struct_open_g_l_graphics_context_custom_shader_a847370481b14927f5571dbb24bf4110b}\label{struct_open_g_l_graphics_context_custom_shader_a847370481b14927f5571dbb24bf4110b}}
