Called when the mouse moves out of a component.

This will be called when the mouse moves off the edge of this component.

If the mouse button was pressed, and it was then dragged off the edge of the component and released, then this callback will happen when the button is released, after the mouse\+Up callback.

Called when the mouse moves inside a component.

If the mouse button isn\textquotesingle{}t pressed and the mouse moves over a component, this will be called to let the component react to this.

A component will always get a mouse\+Enter callback before a mouse\+Move.
