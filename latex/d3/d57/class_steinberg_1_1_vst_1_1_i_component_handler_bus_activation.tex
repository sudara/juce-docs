Allows the plug-\/in to request the host to activate or deactivate a specific bus. If the host accepts this request, it will call later on \mbox{\hyperlink{class_steinberg_1_1_vst_1_1_i_component_ac58e1a76dd961bfee3795994a0a442bc}{IComponent\+::activate\+Bus}}. This is particularly useful for instruments with more than 1 outputs, where the user could request from the plug-\/in UI a given output bus activation.
