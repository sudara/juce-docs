Note that this proprietary format is mainly kept here for backwards-\/compatibility, and you may prefer to use the \mbox{\hyperlink{struct_base64_a25374b3e27e8413191e7567cdc499100}{Base64\+::to\+Base64()}} method if you want to use the standard base-\/64 encoding.
