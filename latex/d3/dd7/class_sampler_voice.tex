\doxysubsection{Detailed Description}
A subclass of \mbox{\hyperlink{class_synthesiser_voice}{Synthesiser\+Voice}} that can play a \mbox{\hyperlink{class_sampler_sound}{Sampler\+Sound}}.

To use it, create a \mbox{\hyperlink{class_synthesiser}{Synthesiser}}, add some \mbox{\hyperlink{class_sampler_voice}{Sampler\+Voice}} objects to it, then give it some Sampled\+Sound objects to play.

Renders the next block of data for this voice.

The output audio data must be added to the current contents of the buffer provided. Only the region of the buffer between start\+Sample and (start\+Sample + num\+Samples) should be altered by this method.

If the voice is currently silent, it should just return without doing anything.

If the sound that the voice is playing finishes during the course of this rendered block, it must call \mbox{\hyperlink{class_synthesiser_voice_a55d1e7a062f8dc9a6cd78526b0da049c}{clear\+Current\+Note()}}, to tell the synthesiser that it has finished.

The size of the blocks that are rendered can change each time it is called, and may involve rendering as little as 1 sample at a time. In between rendering callbacks, the voice\textquotesingle{}s methods will be called to tell it about note and controller events.

{\footnotesize\ttfamily \mbox{\hyperlink{png_8h_aa8c59027f9ab2769342f248709d68d17}{void}} Sampler\+Voice\+::stop\+Note (\begin{DoxyParamCaption}\item[{float}]{velocity,  }\item[{bool}]{allow\+Tail\+Off }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [override]}, {\ttfamily [virtual]}}

Called to stop a note.

This will be called during the rendering callback, so must be fast and thread-\/safe.

The velocity indicates how quickly the note was released -\/ 0 is slowly, 1 is quickly.

If allow\+Tail\+Off is false or the voice doesn\textquotesingle{}t want to tail-\/off, then it must stop all sound immediately, and must call \mbox{\hyperlink{class_synthesiser_voice_a55d1e7a062f8dc9a6cd78526b0da049c}{clear\+Current\+Note()}} to reset the state of this voice and allow the synth to reassign it another sound.

If allow\+Tail\+Off is true and the voice decides to do a tail-\/off, then it\textquotesingle{}s allowed to begin fading out its sound, and it can stop playing until it\textquotesingle{}s finished. As soon as it finishes playing (during the rendering callback), it must make sure that it calls \mbox{\hyperlink{class_synthesiser_voice_a55d1e7a062f8dc9a6cd78526b0da049c}{clear\+Current\+Note()}}.
