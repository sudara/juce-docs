\doxysubsection{Detailed Description}
This stream will decompress a source-\/stream using zlib.

Tip\+: if you\textquotesingle{}re reading lots of small items from one of these streams, you can increase the performance enormously by passing it through a \mbox{\hyperlink{class_buffered_input_stream}{Buffered\+Input\+Stream}}, so that it has to read larger blocks less often.
