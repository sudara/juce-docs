\doxysubsection{Detailed Description}
Controls the order in which keyboard focus moves between components.

The default behaviour of this class uses a \mbox{\hyperlink{class_focus_traverser}{Focus\+Traverser}} object internally to determine the default/next/previous component until it finds one which wants keyboard focus, as set by the \mbox{\hyperlink{class_component_a6a1f21a76b971d54870cb5c32c041055}{Component\+::set\+Wants\+Keyboard\+Focus()}} method.

If you need keyboard focus traversal in a more customised way, you can create a subclass of \mbox{\hyperlink{class_component_traverser}{Component\+Traverser}} that uses your own algorithm, and use \mbox{\hyperlink{class_component_a8dc872ef83fedecd89cf814d28236cd0}{Component\+::create\+Keyboard\+Focus\+Traverser()}} to create it.
