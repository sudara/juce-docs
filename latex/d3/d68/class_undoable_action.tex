If possible, this method should create and return a single action that does the same job as this one followed by the supplied action.

If it\textquotesingle{}s not possible to merge the two actions, the method should return a nullptr.

Returns a value to indicate how much memory this object takes up.

Because the \mbox{\hyperlink{class_undo_manager}{Undo\+Manager}} keeps a list of Undoable\+Actions, this is used to work out how much space each one will take up, so that the \mbox{\hyperlink{class_undo_manager}{Undo\+Manager}} can work out how many to keep.

The default value returned here is 10 -\/ units are arbitrary and don\textquotesingle{}t have to be accurate.
