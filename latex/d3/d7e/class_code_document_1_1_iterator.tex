\doxysubsection{Detailed Description}
Iterates the text in a \mbox{\hyperlink{class_code_document}{Code\+Document}}.

This class lets you read characters from a \mbox{\hyperlink{class_code_document}{Code\+Document}}. It\textquotesingle{}s designed to be used by a \mbox{\hyperlink{class_code_tokeniser}{Code\+Tokeniser}} object.
