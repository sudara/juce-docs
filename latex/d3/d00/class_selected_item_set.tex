\subsubsection*{template$<$class Selectable\+Item\+Type$>$\newline
class Selected\+Item\+Set$<$ Selectable\+Item\+Type $>$}
Manages a list of selectable items.

Use one of these to keep a track of things that the user has highlighted, like icons or things in a list.

The class is templated so that you can use it to hold either a set of pointers to objects, or a set of ID numbers or handles, for cases where each item may not always have a corresponding object.

To be informed when items are selected/deselected, register a \mbox{\hyperlink{class_change_listener}{Change\+Listener}} with this object.

Selects or deselects an item.

This will use the modifier keys to decide whether to deselect other items first.

So if the shift key is held down, the item will be added without deselecting anything (same as calling \mbox{\hyperlink{class_selected_item_set_a8ec6de21f75ccfa083cd702ab027f778}{add\+To\+Selection()}} )

If no modifiers are down, the current selection will be cleared first (same as calling \mbox{\hyperlink{class_selected_item_set_a5c989a8cd1483b655b948aee86fb768c}{select\+Only()}} )

If the ctrl (or command on the Mac) key is held down, the item will be toggled -\/ so it\textquotesingle{}ll be added to the set unless it\textquotesingle{}s already there, in which case it\textquotesingle{}ll be deselected.

If the items that you\textquotesingle{}re selecting can also be dragged, you may need to use the \mbox{\hyperlink{class_selected_item_set_ab37144a1dbc919d2fef3b3e496217b03}{add\+To\+Selection\+On\+Mouse\+Down()}} and \mbox{\hyperlink{class_selected_item_set_a1347c76b84176da73807d6d9883e2cbf}{add\+To\+Selection\+On\+Mouse\+Up()}} calls to handle the subtleties of this kind of usage.

Selects or deselects items that can also be dragged, based on a mouse-\/down event.

If you call \mbox{\hyperlink{class_selected_item_set_ab37144a1dbc919d2fef3b3e496217b03}{add\+To\+Selection\+On\+Mouse\+Down()}} at the start of your mouse\+Down event, and then call \mbox{\hyperlink{class_selected_item_set_a1347c76b84176da73807d6d9883e2cbf}{add\+To\+Selection\+On\+Mouse\+Up()}} at the end of your mouse\+Up event, this makes it easy to handle multiple-\/selection of sets of objects that can also be dragged.

For example, if you have several items already selected, and you click on one of them (without dragging), then you\textquotesingle{}d expect this to deselect the other, and just select the item you clicked on. But if you had clicked on this item and dragged it, you\textquotesingle{}d have expected them all to stay selected.

When you call this method, you\textquotesingle{}ll need to store the boolean result, because the \mbox{\hyperlink{class_selected_item_set_a1347c76b84176da73807d6d9883e2cbf}{add\+To\+Selection\+On\+Mouse\+Up()}} method will need to be know this value.
