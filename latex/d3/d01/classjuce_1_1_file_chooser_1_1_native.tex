When a component is modal, this callback allows it to choose which other components can still receive events.

When a modal component is active and the user clicks on a non-\/modal component, this method is called on the modal component, and if it returns true, the event is allowed to reach its target. If it returns false, the event is blocked and the \mbox{\hyperlink{class_component_a260e46fb72d4b3210bc46a658a21fbb5}{input\+Attempt\+When\+Modal()}} callback is made.

It called by the \mbox{\hyperlink{class_component_a70fc33489ec892d615452dfdbd4687db}{is\+Currently\+Blocked\+By\+Another\+Modal\+Component()}} method. The default implementation just returns false in all cases.
