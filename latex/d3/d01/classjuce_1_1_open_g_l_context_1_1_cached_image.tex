Performs the actual work that this job needs to do.

Your subclass must implement this method, in which is does its work.

If the code in this method takes a significant time to run, it must repeatedly check the \mbox{\hyperlink{class_thread_pool_job_a441a94cba7ec96242fd7268371d9c5ea}{should\+Exit()}} method to see if something is trying to interrupt the job. If \mbox{\hyperlink{class_thread_pool_job_a441a94cba7ec96242fd7268371d9c5ea}{should\+Exit()}} ever returns true, the \mbox{\hyperlink{classjuce_1_1_open_g_l_context_1_1_cached_image_adb413348ef344dbd5f36ae5acaa00871}{run\+Job()}} method must return immediately.

If this method returns job\+Has\+Finished, then the job will be removed from the pool immediately. If it returns job\+Needs\+Running\+Again, then the job will be left in the pool and will get a chance to run again as soon as a thread is free.
