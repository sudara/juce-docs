\subsubsection*{template$<$class Object\+Type$>$\newline
class Linked\+List\+Pointer$<$ Object\+Type $>$}
Helps to manipulate singly-\/linked lists of objects.

For objects that are designed to contain a pointer to the subsequent item in the list, this class contains methods to deal with the list. To use it, the Object\+Type class that it points to must contain a \mbox{\hyperlink{class_linked_list_pointer}{Linked\+List\+Pointer}} called next\+List\+Item, e.\+g.
