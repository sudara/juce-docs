\doxysubsection{Detailed Description}
Acts as an application-\/wide logging class.

A subclass of \mbox{\hyperlink{class_logger}{Logger}} can be created and passed into the \mbox{\hyperlink{class_logger_ad1f9022602d0953f85d67c7075ffb3ed}{Logger\+::set\+Current\+Logger}} method and this will then be used by all calls to write\+To\+Log.

The logger class also contains methods for writing messages to the debugger\textquotesingle{}s output stream.
