\doxysubsection{Detailed Description}
This class is now deprecated in favour of \mbox{\hyperlink{class_midi_buffer_iterator}{Midi\+Buffer\+Iterator}}.

Used to iterate through the events in a \mbox{\hyperlink{class_midi_buffer}{Midi\+Buffer}}.

Note that altering the buffer while an iterator is using it will produce undefined behaviour.
