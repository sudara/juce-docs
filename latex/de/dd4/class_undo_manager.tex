\doxysubsection{Detailed Description}
Manages a list of undo/redo commands.

An \mbox{\hyperlink{class_undo_manager}{Undo\+Manager}} object keeps a list of past actions and can use these actions to move backwards and forwards through an undo history.

To use it, create subclasses of \mbox{\hyperlink{class_undoable_action}{Undoable\+Action}} which perform all the actions you need, then when you need to actually perform an action, create one and pass it to the \mbox{\hyperlink{class_undo_manager}{Undo\+Manager}}\textquotesingle{}s \mbox{\hyperlink{class_undo_manager_a5f8b51dbb061ca3a1a8916938e0e2f49}{perform()}} method.

The manager also uses the concept of \textquotesingle{}transactions\textquotesingle{} to group the actions together -\/ all actions performed between calls to \mbox{\hyperlink{class_undo_manager_a2b60b5f1763023351eddf813d4e7cc78}{begin\+New\+Transaction()}} are grouped together and are all undone/redone as a group.

The \mbox{\hyperlink{class_undo_manager}{Undo\+Manager}} is a \mbox{\hyperlink{class_change_broadcaster}{Change\+Broadcaster}}, so listeners can register to be told when actions are performed or undone.

{\footnotesize\ttfamily bool Undo\+Manager\+::undo\+Current\+Transaction\+Only (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}

Tries to roll-\/back any actions that were added to the current transaction.

This will perform an \mbox{\hyperlink{class_undo_manager_a39f45c284e8d0df1a0d378e676246931}{undo()}} only if there are some actions in the undo list that were added after the last call to \mbox{\hyperlink{class_undo_manager_a2b60b5f1763023351eddf813d4e7cc78}{begin\+New\+Transaction()}}.

This is useful because it lets you call \mbox{\hyperlink{class_undo_manager_a2b60b5f1763023351eddf813d4e7cc78}{begin\+New\+Transaction()}}, then perform an operation which may or may not actually perform some actions, and then call this method to get rid of any actions that might have been done without it rolling back the previous transaction if nothing was actually done.

\begin{DoxyReturn}{Returns}
true if any actions were undone. 
\end{DoxyReturn}
