\doxysubsection{Detailed Description}
The basic interface of all interfaces.


\begin{DoxyItemize}
\item The \mbox{\hyperlink{class_steinberg_1_1_f_unknown_a784e44c90535adbbf7f83ca670e195db}{FUnknown\+::query\+Interface}} method is used to retrieve pointers to other interfaces of the object.
\item \mbox{\hyperlink{class_steinberg_1_1_f_unknown_a318d18517e98fd8115b66b9c64096b75}{FUnknown\+::add\+Ref}} and \mbox{\hyperlink{class_steinberg_1_1_f_unknown_a5e929b90aef2e598f19e41fb8081b6c9}{FUnknown\+::release}} manage the lifetime of the object. If no more references exist, the object is destroyed in memory.
\end{DoxyItemize}

Interfaces are identified by 16 byte Globally Unique Identifiers. The SDK provides a class called \mbox{\hyperlink{class_steinberg_1_1_f_u_i_d}{FUID}} for this purpose.
