\doxysubsection{Detailed Description}
An object of this class maintains a connection between a \mbox{\hyperlink{class_button}{Button}} and a plug-\/in parameter.

During the lifetime of this object it keeps the two things in sync, making it easy to connect a button to a parameter. When this object is deleted, the connection is broken. Make sure that your parameter and \mbox{\hyperlink{class_button}{Button}} are not deleted before this object!
