\doxysubsection{Detailed Description}
A path is a sequence of lines and curves that may either form a closed shape or be open-\/ended.

To use a path, you can create an empty one, then add lines and curves to it to create shapes, then it can be rendered by a \mbox{\hyperlink{class_graphics}{Graphics}} context or used for geometric operations.

e.\+g. 
\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{class_path}{Path}} myPath;}
\DoxyCodeLine{}
\DoxyCodeLine{myPath.\mbox{\hyperlink{class_path_a9c24222b2226b0e082069dbf789dca07}{startNewSubPath}} (10.0f, 10.0f);          \textcolor{comment}{// move the current position to (10, 10)}}
\DoxyCodeLine{myPath.\mbox{\hyperlink{class_path_a6cfdf1dea900089ffee8eea61385062e}{lineTo}} (100.0f, 200.0f);                 \textcolor{comment}{// draw a line from here to (100, 200)}}
\DoxyCodeLine{myPath.\mbox{\hyperlink{class_path_ab50510cf385b9ea3fe64fef6be45f85b}{quadraticTo}} (0.0f, 150.0f, 5.0f, 50.0f); \textcolor{comment}{// draw a curve that ends at (5, 50)}}
\DoxyCodeLine{myPath.\mbox{\hyperlink{class_path_a3fb7879fd6e09ce7657260c7b2934162}{closeSubPath}}();                          \textcolor{comment}{// close the subpath with a line back to (10, 10)}}
\DoxyCodeLine{}
\DoxyCodeLine{\textcolor{comment}{// add an ellipse as well, which will form a second sub-\/path within the path..}}
\DoxyCodeLine{myPath.\mbox{\hyperlink{class_path_a7514c5eaa928b64121490a7f0ce3088c}{addEllipse}} (50.0f, 50.0f, 40.0f, 30.0f);}
\DoxyCodeLine{}
\DoxyCodeLine{\textcolor{comment}{// double the width of the whole thing..}}
\DoxyCodeLine{myPath.\mbox{\hyperlink{class_path_afad2558a4296d94b9f5640281ec3c0c3}{applyTransform}} (\mbox{\hyperlink{class_affine_transform_a93fbc1d4a28ba08c5542db932e78f9a7}{AffineTransform::scale}} (2.0f, 1.0f));}
\DoxyCodeLine{}
\DoxyCodeLine{\textcolor{comment}{// and draw it to a graphics context with a 5-\/pixel thick outline.}}
\DoxyCodeLine{\mbox{\hyperlink{namespacegl_a27efffdbb9aab3859a7f9c17ea0ae26e}{g}}.strokePath (myPath, \mbox{\hyperlink{class_path_stroke_type}{PathStrokeType}} (5.0f));}

\end{DoxyCode}


A path object can actually contain multiple sub-\/paths, which may themselves be open or closed.

Preallocates enough space for adding the given number of coordinates to the path. Changes the winding-\/rule to be used when filling the path.

If set to true (which is the default), then the path uses a non-\/zero-\/winding rule to determine which points are inside the path. If set to false, it uses an alternate-\/winding rule.

The winding-\/rule comes into play when areas of the shape overlap other areas, and determines whether the overlapping regions are considered to be inside or outside.

Changing this value just sets a flag -\/ it doesn\textquotesingle{}t affect the contents of the path.
