\doxysubsection{Detailed Description}
Interface class for graphics context objects, used internally by the \mbox{\hyperlink{class_graphics}{Graphics}} class.

Users are not supposed to create instances of this class directly -\/ do your drawing via the \mbox{\hyperlink{class_graphics}{Graphics}} object instead.

It\textquotesingle{}s a base class for different types of graphics context, that may perform software-\/based or OS-\/accelerated rendering.

E.\+g. the \mbox{\hyperlink{class_low_level_graphics_software_renderer}{Low\+Level\+Graphics\+Software\+Renderer}} renders onto an image in memory, but other subclasses could render directly to a windows HDC, a Quartz context, or an Open\+GL context.
