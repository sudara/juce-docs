\doxysubsection{Detailed Description}
Manages a collection of properties.

This is a slightly higher-\/level wrapper for managing \mbox{\hyperlink{class_properties_file}{Properties\+File}} objects.

It holds two different \mbox{\hyperlink{class_properties_file}{Properties\+File}} objects internally, one for user-\/specific settings (stored in your user directory), and one for settings that are common to all users (stored in a folder accessible to all users).

The class manages the creation of these files on-\/demand, allowing access via the \mbox{\hyperlink{class_application_properties_acefac770672d761f5a8d0fc33e4f4d3c}{get\+User\+Settings()}} and \mbox{\hyperlink{class_application_properties_af8ebec199c2f541fdea287f28ebc293e}{get\+Common\+Settings()}} methods.

After creating an instance of an \mbox{\hyperlink{class_application_properties}{Application\+Properties}} object, you should first of all call \mbox{\hyperlink{class_application_properties_ab8f0b0ee2ece06a23d36996eaa1a7107}{set\+Storage\+Parameters()}} to tell it the parameters to use to create its files.

Note that when you search the user \mbox{\hyperlink{class_properties_file}{Properties\+File}} for a value that it doesn\textquotesingle{}t contain, the common settings are used as a second-\/chance place to look.
