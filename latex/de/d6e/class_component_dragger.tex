\doxysubsection{Detailed Description}
An object to take care of the logic for dragging components around with the mouse.

Very easy to use -\/ in your mouse\+Down() callback, call \mbox{\hyperlink{class_component_dragger_a3a6ae550729873ec7fa31a48bc79a352}{start\+Dragging\+Component()}}, then in your mouse\+Drag() callback, call \mbox{\hyperlink{class_component_dragger_af2ebcd662f3279b373bfaf035f2c269e}{drag\+Component()}}.

When starting a drag, you can give it a \mbox{\hyperlink{class_component_bounds_constrainer}{Component\+Bounds\+Constrainer}} to use to limit the component\textquotesingle{}s position and keep it on-\/screen.
