\doxysubsection{Detailed Description}
An implementation of the \mbox{\hyperlink{class_online_unlock_status}{Online\+Unlock\+Status}} class which talks to the Tracktion Marketplace server.

For details about how to use this class, see the docs for the base class\+: \mbox{\hyperlink{class_online_unlock_status}{Online\+Unlock\+Status}}. Basically, you need to inherit from it, and implement all the pure virtual methods to tell it about your product.
