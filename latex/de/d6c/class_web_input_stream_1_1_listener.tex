\doxysubsection{Detailed Description}
Used to receive callbacks for POST data send progress.

Pass one of these into the \mbox{\hyperlink{class_web_input_stream_abe8920cdc15843c44228c267db3e85db}{connect()}} method and its \mbox{\hyperlink{class_web_input_stream_1_1_listener_ae7398552087db4dc1b1f8ca14ecaacd9}{post\+Data\+Send\+Progress()}} method will be called periodically with updates on POST data upload progress.
