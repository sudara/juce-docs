\doxysubsection{Detailed Description}
Used to build a tree of elements representing an XML document.

An XML document can be parsed into a tree of Xml\+Elements, each of which represents an XML tag structure, and which may itself contain other nested elements.

An \mbox{\hyperlink{class_xml_element}{Xml\+Element}} can also be converted back into a text document, and has lots of useful methods for manipulating its attributes and sub-\/elements, so Xml\+Elements can actually be used as a handy general-\/purpose data structure.

Here\textquotesingle{}s an example of parsing some elements\+:
