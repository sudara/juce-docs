\doxysubsection{Detailed Description}


Description of a chord. A chord is described with a key note, a root note and the Bitmask of a chord. ~\newline
1st bit set\+: minor second; 2nd bit set\+: major second, and so on. ~\newline
There is {\bfseries{no}} bit for the keynote (root of the chord) because it is inherently always present. ~\newline
