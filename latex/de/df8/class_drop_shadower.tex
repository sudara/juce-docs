\doxysubsection{Detailed Description}
Adds a drop-\/shadow to a component.

This object creates and manages a set of components which sit around a component, creating a gaussian shadow around it. The components will track the position of the component and if it\textquotesingle{}s brought to the front they\textquotesingle{}ll also follow this.

For desktop windows you don\textquotesingle{}t need to use this class directly -\/ just set the Component\+::window\+Has\+Drop\+Shadow flag when calling \mbox{\hyperlink{class_component_ad717da76012d971197fd8cf943f3e721}{Component\+::add\+To\+Desktop()}}, and the system will create one of these if it\textquotesingle{}s needed (which it obviously isn\textquotesingle{}t on the Mac, for example).
