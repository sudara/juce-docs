This is the character encoding type used internally to store the string.

By setting the value of JUCE\+\_\+\+STRING\+\_\+\+UTF\+\_\+\+TYPE to 8, 16, or 32, you can change the internal storage format of the \mbox{\hyperlink{class_string}{String}} class. UTF-\/8 uses the least space (if your strings contain few extended characters), but call operator\mbox{[}\mbox{]} involves iterating the string to find the required index. UTF-\/32 provides instant random access to its characters, but uses 4 bytes per character to store them.
