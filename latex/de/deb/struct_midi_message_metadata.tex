\doxysubsection{Detailed Description}
A view of MIDI message data stored in a contiguous buffer.

Instances of this class do {\itshape not} own the midi data bytes that they point to. Instead, they expect the midi data to live in a separate buffer that outlives the \mbox{\hyperlink{struct_midi_message_metadata}{Midi\+Message\+Metadata}} instance.
