\doxysubsection{Detailed Description}
A class for storing and manipulating a source code file.

When using a \mbox{\hyperlink{class_code_editor_component}{Code\+Editor\+Component}}, it takes one of these as its source object.

The \mbox{\hyperlink{class_code_document}{Code\+Document}} stores its content as an array of lines, which makes it quick to insert and delete.
