\doxysubsection{Detailed Description}
Classes can implement this interface and register themselves with the \mbox{\hyperlink{class_desktop}{Desktop}} class to receive callbacks when the operating system dark mode setting changes. The \mbox{\hyperlink{class_desktop_a33ca1091178824b63a8e8eaf14f3e7bb}{Desktop\+::is\+Dark\+Mode\+Active()}} method can then be used to query the current setting.
