\doxysubsection{Detailed Description}
A Pre-\/formatted piece of text, which may contain multiple fonts and colours.

A \mbox{\hyperlink{class_text_layout}{Text\+Layout}} is created from an \mbox{\hyperlink{class_attributed_string}{Attributed\+String}}, and once created can be quickly drawn into a \mbox{\hyperlink{class_graphics}{Graphics}} context.
