\doxysubsection{Detailed Description}
An interface to allow an \mbox{\hyperlink{class_audio_processor}{Audio\+Processor}} to send and receive VST specific calls from the host.

To use this class, ensure that your \mbox{\hyperlink{class_audio_processor}{Audio\+Processor}} publicly inherits from \mbox{\hyperlink{struct_v_s_t_callback_handler}{VSTCallback\+Handler}}.
