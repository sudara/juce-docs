\doxysubsection{Detailed Description}
This class acts as a wrapper around a property inside a \mbox{\hyperlink{class_value_tree}{Value\+Tree}}.

If the property inside the \mbox{\hyperlink{class_value_tree}{Value\+Tree}} is missing it will return a default value, which can be specified in the constructor or by calling \mbox{\hyperlink{class_value_tree_property_with_default_a46c4e34e56538db22b0b2aaca091e7af}{set\+Default()}}.
