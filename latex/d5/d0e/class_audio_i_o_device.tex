\doxysubsection{Detailed Description}
Base class for an audio device with synchronised input and output channels.

Subclasses of this are used to implement different protocols such as Direct\+Sound, ASIO, Core\+Audio, etc.

To create one of these, you\textquotesingle{}ll need to use the \mbox{\hyperlink{class_audio_i_o_device_type}{Audio\+IODevice\+Type}} class -\/ see the documentation for that class for more info.

For an easier way of managing audio devices and their settings, have a look at the \mbox{\hyperlink{class_audio_device_manager}{Audio\+Device\+Manager}} class.
