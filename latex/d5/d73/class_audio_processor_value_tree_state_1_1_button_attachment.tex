\doxysubsection{Detailed Description}
An object of this class maintains a connection between a \mbox{\hyperlink{class_button}{Button}} and a parameter in an \mbox{\hyperlink{class_audio_processor_value_tree_state}{Audio\+Processor\+Value\+Tree\+State}}.

During the lifetime of this \mbox{\hyperlink{class_audio_processor_value_tree_state_1_1_button_attachment}{Button\+Attachment}} object, it keeps the two things in sync, making it easy to connect a button to a parameter. When this object is deleted, the connection is broken. Make sure that your \mbox{\hyperlink{class_audio_processor_value_tree_state}{Audio\+Processor\+Value\+Tree\+State}} and \mbox{\hyperlink{class_button}{Button}} aren\textquotesingle{}t deleted before this object!
