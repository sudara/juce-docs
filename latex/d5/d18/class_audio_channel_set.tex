\doxysubsection{Detailed Description}
Represents a set of audio channel types.

For example, you might have a set of left + right channels, which is a stereo channel set. It is a collection of values from the \mbox{\hyperlink{class_audio_channel_set_ab917e490df2e0edfb1087e4c4f8d652e}{Audio\+Channel\+Set\+::\+Channel\+Type}} enum, where each type may only occur once within the set.

The documentation below lists which \mbox{\hyperlink{class_audio_channel_set}{Audio\+Channel\+Set}} corresponds to which native layouts used by AAX, VST2/\+VST3 and Core\+Audio/\+AU. The layout tags in Core\+Audio are particularly confusing. For example, the layout which is labeled as \char`\"{}7.\+1 SDDS\char`\"{} in Logic Pro, corresponds to Core\+Audio/\+AU\textquotesingle{}s k\+Audio\+Channel\+Layout\+Tag\+\_\+\+DTS\+\_\+7\+\_\+0 tag, whereas AAX\textquotesingle{}s DTS 7.\+1 Layout corresponds to Core\+Audio/\+AU\textquotesingle{}s k\+Audio\+Channel\+Layout\+Tag\+\_\+\+MPEG\+\_\+7\+\_\+1\+\_\+A format, etc. Please do not use the Core\+Audio tag as an indication to the actual layout of the speakers.
