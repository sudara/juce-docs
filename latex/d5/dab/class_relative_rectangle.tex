\doxysubsection{Detailed Description}
A rectangle stored as a set of \mbox{\hyperlink{class_relative_coordinate}{Relative\+Coordinate}} values.

The rectangle\textquotesingle{}s top, left, bottom and right edge positions are each stored as a \mbox{\hyperlink{class_relative_coordinate}{Relative\+Coordinate}}.
