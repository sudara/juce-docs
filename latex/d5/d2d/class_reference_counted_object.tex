\doxysubsection{Detailed Description}
A base class which provides methods for reference-\/counting.

To add reference-\/counting to a class, derive it from this class, and use the \mbox{\hyperlink{class_reference_counted_object_ptr}{Reference\+Counted\+Object\+Ptr}} class to point to it.

e.\+g. 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keyword}{class }MyClass : \textcolor{keyword}{public} \mbox{\hyperlink{class_reference_counted_object}{ReferenceCountedObject}}}
\DoxyCodeLine{\{}
\DoxyCodeLine{    \textcolor{keywordtype}{void} foo();}
\DoxyCodeLine{}
\DoxyCodeLine{    \textcolor{comment}{// This is a neat way of declaring a typedef for a pointer class,}}
\DoxyCodeLine{    \textcolor{comment}{// rather than typing out the full templated name each time..}}
\DoxyCodeLine{    \textcolor{keyword}{using} Ptr = \mbox{\hyperlink{class_reference_counted_object_ptr}{ReferenceCountedObjectPtr<MyClass>}};}
\DoxyCodeLine{\};}
\DoxyCodeLine{}
\DoxyCodeLine{MyClass::Ptr \mbox{\hyperlink{namespacegl_ab8b154a639177df172c80eddeb09a533}{p}} = \textcolor{keyword}{new} MyClass();}
\DoxyCodeLine{MyClass::Ptr p2 = \mbox{\hyperlink{namespacegl_ab8b154a639177df172c80eddeb09a533}{p}};}
\DoxyCodeLine{\mbox{\hyperlink{namespacegl_ab8b154a639177df172c80eddeb09a533}{p}} = \textcolor{keyword}{nullptr};}
\DoxyCodeLine{p2-\/>foo();}

\end{DoxyCode}


Once a new \mbox{\hyperlink{class_reference_counted_object}{Reference\+Counted\+Object}} has been assigned to a pointer, be careful not to delete the object manually.

This class uses an Atomic$<$int$>$ value to hold the reference count, so the reference count can be updated on multiple threads. Note that whilst it\textquotesingle{}s thread-\/safe to create and delete a \mbox{\hyperlink{class_reference_counted_object_ptr}{Reference\+Counted\+Object\+Ptr}} to a \mbox{\hyperlink{class_reference_counted_object}{Reference\+Counted\+Object}} shared between threads, it\textquotesingle{}s not thread-\/safe to modify or swap the \mbox{\hyperlink{class_reference_counted_object}{Reference\+Counted\+Object}}.

For a faster but non-\/thread-\/safe version, use \mbox{\hyperlink{class_single_threaded_reference_counted_object}{Single\+Threaded\+Reference\+Counted\+Object}} instead.
