Callback to indicate that the user has dropped the files onto this component.

When the user drops the files, this get called, and you can use the files in whatever way is appropriate.

Note that after this is called, the file\+Drag\+Exit method may not be called, so you should clean up in here if there\textquotesingle{}s anything you need to do when the drag finishes.
