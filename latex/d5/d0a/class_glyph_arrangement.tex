\doxysubsection{Detailed Description}
A set of glyphs, each with a position.

You can create a \mbox{\hyperlink{class_glyph_arrangement}{Glyph\+Arrangement}}, text to it and then draw it onto a graphics context. It\textquotesingle{}s used internally by the text methods in the \mbox{\hyperlink{class_graphics}{Graphics}} class, but can be used directly if more control is needed.

Tries to fit some text within a given space.

This does its best to make the given text readable within the specified rectangle, so it\textquotesingle{}s useful for labelling things.

If the text is too big, it\textquotesingle{}ll be squashed horizontally or broken over multiple lines if the maximum\+Lines\+To\+Use value allows this. If the text just won\textquotesingle{}t fit into the space, it\textquotesingle{}ll cram as much as possible in there, and put some ellipsis at the end to show that it\textquotesingle{}s been truncated.

A \mbox{\hyperlink{class_justification}{Justification}} parameter lets you specify how the text is laid out within the rectangle, both horizontally and vertically.

The minimum\+Horizontal\+Scale parameter specifies how much the text can be squashed horizontally to try to squeeze it into the space. If you don\textquotesingle{}t want any horizontal scaling to occur, you can set this value to 1.\+0f. Pass 0 if you want it to use the default value.

Adds some multi-\/line text, breaking lines at word-\/boundaries if they are too wide.

This will add text to the arrangement, breaking it into new lines either where there is a new-\/line or carriage-\/return character in the text, or where a line\textquotesingle{}s width exceeds the value set in max\+Line\+Width.

Each line that is added will be laid out using the flags set in horizontal\+Layout, so the lines can be left-\/ or right-\/justified, or centred horizontally in the space between x and (x + max\+Line\+Width).

The y coordinate is the position of the baseline of the first line of text -\/ subsequent lines will be placed below it, separated by a distance of font.\+get\+Height() + leading. \mbox{\Hypertarget{class_glyph_arrangement_a0ace103d7c4b6600dcea7aff1f26811d}\label{class_glyph_arrangement_a0ace103d7c4b6600dcea7aff1f26811d}}
