\doxysubsection{Detailed Description}
A set of routines to convert buffers of 32-\/bit floating point data to and from various integer formats.

Note that these functions are deprecated -\/ the \mbox{\hyperlink{class_audio_data}{Audio\+Data}} class provides a much more flexible set of conversion classes now.
