\doxysubsection{Detailed Description}
A component that displays an MPE-\/compatible keyboard, whose notes can be clicked on.

This component will mimic a physical MPE-\/compatible keyboard, showing the current state of an \mbox{\hyperlink{class_m_p_e_instrument}{MPEInstrument}} object. When the on-\/screen keys are clicked on, it will play these notes by calling the note\+On() and note\+Off() methods of its \mbox{\hyperlink{class_m_p_e_instrument}{MPEInstrument}} object. Moving the mouse will update the pitchbend and timbre dimensions of the \mbox{\hyperlink{class_m_p_e_instrument}{MPEInstrument}}.
