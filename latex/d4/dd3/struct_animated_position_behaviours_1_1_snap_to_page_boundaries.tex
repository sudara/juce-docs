\doxysubsection{Detailed Description}
A behaviour that gravitates an \mbox{\hyperlink{class_animated_position}{Animated\+Position}} object towards the nearest integer position when released.

This class is intended to be used as a template parameter to the \mbox{\hyperlink{class_animated_position}{Animated\+Position}} class. It\textquotesingle{}s handy when using an \mbox{\hyperlink{class_animated_position}{Animated\+Position}} to show a series of pages, because it allows the pages can be scrolled smoothly, but when released, snaps back to show a whole page.
