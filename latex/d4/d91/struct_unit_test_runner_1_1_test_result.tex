\doxysubsection{Detailed Description}
Contains the results of a test.

One of these objects is instantiated each time \mbox{\hyperlink{class_unit_test_ae21d8c58358819742f379706b6b34753}{Unit\+Test\+::begin\+Test()}} is called, and it contains details of the number of subsequent \mbox{\hyperlink{class_unit_test_a83e2912ee18731e0bfaf1c21a50b1131}{Unit\+Test\+::expect()}} calls that are made.
