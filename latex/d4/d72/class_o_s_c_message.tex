\doxysubsection{Detailed Description}
An OSC \mbox{\hyperlink{class_message}{Message}}.

An \mbox{\hyperlink{class_o_s_c_message}{OSCMessage}} consists of an \mbox{\hyperlink{class_o_s_c_address_pattern}{OSCAddress\+Pattern}} and zero or more OSCArguments.

OSC messages are the elementary objects that are used to exchange any data via OSC. An \mbox{\hyperlink{class_o_s_c_sender}{OSCSender}} can send \mbox{\hyperlink{class_o_s_c_message}{OSCMessage}} objects to an \mbox{\hyperlink{class_o_s_c_receiver}{OSCReceiver}}.
