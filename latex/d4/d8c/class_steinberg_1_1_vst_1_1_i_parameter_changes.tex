\doxysubsection{Detailed Description}
All parameter changes of a processing block\+: \mbox{\hyperlink{class_steinberg_1_1_vst_1_1_i_parameter_changes}{Vst\+::\+IParameter\+Changes}}


\begin{DoxyItemize}

\item \mbox{[}released\+: 3.\+0.\+0\mbox{]}
\item \mbox{[}mandatory\mbox{]}

This interface is used to transmit any changes to be applied to parameters in the current processing block. A change can be caused by GUI interaction as well as automation. They are transmitted as a list of queues (\mbox{\hyperlink{class_steinberg_1_1_vst_1_1_i_param_value_queue}{IParam\+Value\+Queue}}) containing only queues for parameters that actually did change.
