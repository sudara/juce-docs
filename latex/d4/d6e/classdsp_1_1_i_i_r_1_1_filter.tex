\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Sample\+Type$>$\newline
class dsp\+::\+IIR\+::\+Filter$<$ Sample\+Type $>$}
A processing class that can perform \mbox{\hyperlink{namespacedsp_1_1_i_i_r}{IIR}} filtering on an audio signal, using the Transposed Direct Form II digital structure.

If you need a lowpass, bandpass or highpass filter with fast modulation of its cutoff frequency, you might use the class \mbox{\hyperlink{namespacedsp_1_1_state_variable_filter}{State\+Variable\+Filter}} instead, which is designed to prevent artefacts at parameter changes, instead of the class \mbox{\hyperlink{classdsp_1_1_i_i_r_1_1_filter}{Filter}}.
