\doxysubsection{Detailed Description}
The base class for objects that can be sent to a \mbox{\hyperlink{class_message_listener}{Message\+Listener}}.

If you want to send a message that carries some kind of custom data, just create a subclass of \mbox{\hyperlink{class_message}{Message}} with some appropriate member variables to hold your data.

Always create a new instance of a \mbox{\hyperlink{class_message}{Message}} object on the heap, as it will be deleted automatically after the message has been delivered.
