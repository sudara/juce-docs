\doxysubsection{Detailed Description}
Represents a dynamically implemented object.

This class is primarily intended for wrapping scripting language objects, but could be used for other purposes.

An instance of a \mbox{\hyperlink{class_dynamic_object}{Dynamic\+Object}} can be used to store named properties, and by subclassing \mbox{\hyperlink{class_dynamic_object_a5a58c707a0e4aaba522ba2c222cd86f4}{has\+Method()}} and \mbox{\hyperlink{class_dynamic_object_a0a9e1d1438d7ab1cc0aad663137b7908}{invoke\+Method()}}, you can give your object methods.
