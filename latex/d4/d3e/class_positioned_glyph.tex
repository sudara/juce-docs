\doxysubsection{Detailed Description}
A glyph from a particular font, with a particular size, style, typeface and position.

You should rarely need to use this class directly -\/ for most purposes, the \mbox{\hyperlink{class_glyph_arrangement}{Glyph\+Arrangement}} class will do what you need for text layout.
