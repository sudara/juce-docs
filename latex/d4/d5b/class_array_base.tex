\subsubsection*{template$<$class Element\+Type, class Type\+Of\+Critical\+Section\+To\+Use$>$\newline
class Array\+Base$<$ Element\+Type, Type\+Of\+Critical\+Section\+To\+Use $>$}
A basic object container.

This class isn\textquotesingle{}t really for public use -\/ it\textquotesingle{}s used by the other array classes, but might come in handy for some purposes.

It inherits from a critical section class to allow the arrays to use the \char`\"{}empty base class optimisation\char`\"{} pattern to reduce their footprint.
