\doxysubsection{Detailed Description}
Acts as a polymorphic base class for processors. This exposes the same set of methods that a processor must implement as virtual methods, so that you can use the \mbox{\hyperlink{structdsp_1_1_processor_wrapper}{Processor\+Wrapper}} class to wrap an instance of a subclass, and then pass that around using \mbox{\hyperlink{structdsp_1_1_processor_base}{Processor\+Base}} as a base class. \begin{DoxySeeAlso}{See also}

\end{DoxySeeAlso}
