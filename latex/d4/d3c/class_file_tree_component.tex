\doxysubsection{Detailed Description}
A component that displays the files in a directory as a treeview.

This implements the \mbox{\hyperlink{class_directory_contents_display_component}{Directory\+Contents\+Display\+Component}} base class so that it can be used in a \mbox{\hyperlink{class_file_browser_component}{File\+Browser\+Component}}.

To attach a listener to it, use its \mbox{\hyperlink{class_directory_contents_display_component}{Directory\+Contents\+Display\+Component}} base class and the \mbox{\hyperlink{class_file_browser_listener}{File\+Browser\+Listener}} class.
