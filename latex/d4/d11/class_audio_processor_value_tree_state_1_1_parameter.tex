Returns the number of steps that this parameter\textquotesingle{}s range should be quantised into.

If you want a continuous range of values, don\textquotesingle{}t override this method, and allow the default implementation to return \mbox{\hyperlink{class_audio_processor_a143d29f1ad6d9da550d17e38f7fa4607}{Audio\+Processor\+::get\+Default\+Num\+Parameter\+Steps()}}.

If your parameter is boolean, then you may want to make this return 2.

The value that is returned may or may not be used, depending on the host. If you want the host to display stepped automation values, rather than a continuous interpolation between successive values, you should override is\+Discrete to return true.
