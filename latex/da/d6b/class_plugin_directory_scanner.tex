\doxysubsection{Detailed Description}
Scans a directory for plugins, and adds them to a \mbox{\hyperlink{class_known_plugin_list}{Known\+Plugin\+List}}.

To use one of these, create it and call \mbox{\hyperlink{class_plugin_directory_scanner_a5303b7b743f22a6ccd470422f56ecf1e}{scan\+Next\+File()}} repeatedly, until it returns false.

Tries the next likely-\/looking file.

If dont\+Rescan\+If\+Already\+In\+List is true, then the file will only be loaded and re-\/tested if it\textquotesingle{}s not already in the list, or if the file\textquotesingle{}s modification time has changed since the list was created. If dont\+Rescan\+If\+Already\+In\+List is false, the file will always be reloaded and tested. The name\+Of\+Plugin\+Being\+Scanned will be updated to the name of the plugin being scanned before the scan starts.
