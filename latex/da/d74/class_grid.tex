\doxysubsection{Detailed Description}
Container that handles geometry for grid layouts (fixed columns and rows) using a set of declarative rules.

Implemented from the {\ttfamily CSS \mbox{\hyperlink{class_grid}{Grid}} Layout} specification as described at\+: \href{https://css-tricks.com/snippets/css/complete-guide-grid/}{\texttt{ https\+://css-\/tricks.\+com/snippets/css/complete-\/guide-\/grid/}}
