\doxysubsection{Detailed Description}
Describes a type of stroke used to render a solid outline along a path.

A \mbox{\hyperlink{class_path_stroke_type}{Path\+Stroke\+Type}} object can be used directly to create the shape of an outline around a path, and is used by \mbox{\hyperlink{class_graphics_a5c94dcee04df3c4ecd1ebcdda96c043f}{Graphics\+::stroke\+Path}} to specify the type of stroke to draw.
