\doxysubsection{Detailed Description}
An OSC bundle.

An \mbox{\hyperlink{class_o_s_c_bundle}{OSCBundle}} contains an \mbox{\hyperlink{class_o_s_c_time_tag}{OSCTime\+Tag}} and zero or more \mbox{\hyperlink{class_o_s_c_bundle}{OSCBundle}} Elements. The elements of a bundle can be OSC messages or other OSC bundles (this means that OSC bundles can be nested).

This is an advanced OSC structure useful to bundle OSC messages together whose effects must occur simultaneously at some given time. For most use cases it is probably enough to send and receive plain OSC messages.
