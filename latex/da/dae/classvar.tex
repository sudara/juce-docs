\doxysubsection{Detailed Description}
A variant class, that can be used to hold a range of primitive values.

A var object can hold a range of simple primitive values, strings, or any kind of \mbox{\hyperlink{class_reference_counted_object}{Reference\+Counted\+Object}}. The var class is intended to act like the kind of values used in dynamic scripting languages.

You can save/load var objects either in a small, proprietary binary format using \mbox{\hyperlink{classvar_a75dd282271fe372c81c2511d4675c528}{write\+To\+Stream()}}/read\+From\+Stream(), or as \mbox{\hyperlink{class_j_s_o_n}{JSON}} by using the \mbox{\hyperlink{class_j_s_o_n}{JSON}} class.
