\doxysubsection{Detailed Description}
Edit controller extension to describe the plug-\/in structure\+: \mbox{\hyperlink{class_steinberg_1_1_vst_1_1_i_unit_info}{Vst\+::\+IUnit\+Info}}


\begin{DoxyItemize}
\item \mbox{[}plug imp\mbox{]}
\item \mbox{[}extends \mbox{\hyperlink{class_steinberg_1_1_vst_1_1_i_edit_controller}{IEdit\+Controller}}\mbox{]}
\item \mbox{[}released\+: 3.\+0.\+0\mbox{]}
\item \mbox{[}optional\mbox{]}
\end{DoxyItemize}

\mbox{\hyperlink{class_steinberg_1_1_vst_1_1_i_unit_info}{IUnit\+Info}} describes the internal structure of the plug-\/in.
\begin{DoxyItemize}
\item The root unit is the component itself, so get\+Unit\+Count must return 1 at least.
\item The root unit id has to be 0 (k\+Root\+Unit\+Id).
\item Each unit can reference one program list -\/ this reference must not change.
\item Each unit, using a program list, references one program of the list.
\end{DoxyItemize}
