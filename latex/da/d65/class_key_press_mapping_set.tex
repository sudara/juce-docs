\doxysubsection{Detailed Description}
Manages and edits a list of keypresses, which it uses to invoke the appropriate command in an \mbox{\hyperlink{class_application_command_manager}{Application\+Command\+Manager}}.

Normally, you won\textquotesingle{}t actually create a \mbox{\hyperlink{class_key_press_mapping_set}{Key\+Press\+Mapping\+Set}} directly, because each \mbox{\hyperlink{class_application_command_manager}{Application\+Command\+Manager}} contains its own \mbox{\hyperlink{class_key_press_mapping_set}{Key\+Press\+Mapping\+Set}}, so typically you\textquotesingle{}d create yourself an \mbox{\hyperlink{class_application_command_manager}{Application\+Command\+Manager}}, and call its \mbox{\hyperlink{class_application_command_manager_acaab9b68b87978372c18a29d18864892}{Application\+Command\+Manager\+::get\+Key\+Mappings()}} method to get a pointer to its \mbox{\hyperlink{class_key_press_mapping_set}{Key\+Press\+Mapping\+Set}}.

For one of these to actually use keypresses, you\textquotesingle{}ll need to add it as a \mbox{\hyperlink{class_key_listener}{Key\+Listener}} to the top-\/level component for which you want to handle keystrokes. So for example\+:

Normally, you won\textquotesingle{}t actually create a \mbox{\hyperlink{class_key_press_mapping_set}{Key\+Press\+Mapping\+Set}} directly, because each \mbox{\hyperlink{class_application_command_manager}{Application\+Command\+Manager}} contains its own \mbox{\hyperlink{class_key_press_mapping_set}{Key\+Press\+Mapping\+Set}}, so the best thing to do is to create your \mbox{\hyperlink{class_application_command_manager}{Application\+Command\+Manager}}, and use the \mbox{\hyperlink{class_application_command_manager_acaab9b68b87978372c18a29d18864892}{Application\+Command\+Manager\+::get\+Key\+Mappings()}} method to access its mappings.

When a suitable keypress happens, the manager\textquotesingle{}s invoke() method will be used to invoke the appropriate command.

Tries to recreate the mappings from a previously stored state.

The XML passed in must have been created by the \mbox{\hyperlink{class_key_press_mapping_set_a36fea8e522fbecda4ae67bbfe3a9037f}{create\+Xml()}} method.

If the stored state makes any reference to commands that aren\textquotesingle{}t currently available, these will be ignored.

If the set of mappings being loaded was a set of differences (using create\+Xml (true)), then this will call \mbox{\hyperlink{class_key_press_mapping_set_a11ec4b86290489da987e2fcc3455b140}{reset\+To\+Default\+Mappings()}} and then merge the saved mappings on top. If the saved set was created with create\+Xml (false), then this method will first clear all existing mappings and load the saved ones as a complete set.
