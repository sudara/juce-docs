\doxysubsection{Detailed Description}
A base class for a component that goes in a \mbox{\hyperlink{class_property_panel}{Property\+Panel}} and displays one of an item\textquotesingle{}s properties.

Subclasses of this are used to display a property in various forms, e.\+g. a \mbox{\hyperlink{class_choice_property_component}{Choice\+Property\+Component}} shows its value as a combo box; a \mbox{\hyperlink{class_slider_property_component}{Slider\+Property\+Component}} shows its value as a slider; a \mbox{\hyperlink{class_text_property_component}{Text\+Property\+Component}} as a text box, etc.

A subclass must implement the \mbox{\hyperlink{class_property_component_a0ce62ead3836df49c63e253cf33f83c0}{refresh()}} method which will be called to tell the component to update itself, and is also responsible for calling this it when the item that it refers to is changed.

{\footnotesize\ttfamily virtual \mbox{\hyperlink{png_8h_aa8c59027f9ab2769342f248709d68d17}{void}} Property\+Component\+::refresh (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [pure virtual]}}

Updates the property component if the item it refers to has changed.

A subclass must implement this method, and other objects may call it to force it to refresh itself.

The subclass should be economical in the amount of work is done, so for example it should check whether it really needs to do a repaint rather than just doing one every time this method is called, as it may be called when the value being displayed hasn\textquotesingle{}t actually changed.
