\doxysubsection{Detailed Description}
Controls the order in which focus moves between components.

The algorithm used by this class to work out the order of traversal is as follows\+:
\begin{DoxyItemize}
\item Only visible and enabled components are considered focusable.
\item If two components both have an explicit focus order specified then the one with the lowest number comes first (see the \mbox{\hyperlink{class_component_acea5b1758143ff5815fd13ae57f2172e}{Component\+::set\+Explicit\+Focus\+Order()}} method).
\item Any component with an explicit focus order greater than 0 comes before ones that don\textquotesingle{}t have an order specified.
\item Components with their \textquotesingle{}always on top\textquotesingle{} flag set come before those without.
\item Any unspecified components are traversed in a left-\/to-\/right, then top-\/to-\/bottom order.
\end{DoxyItemize}

If you need focus traversal in a more customised way you can create a \mbox{\hyperlink{class_component_traverser}{Component\+Traverser}} subclass that uses your own algorithm and return it from \mbox{\hyperlink{class_component_a138dc2e3c7f4836d7c127409dab8a56d}{Component\+::create\+Focus\+Traverser()}}.
