\doxysubsection{Detailed Description}
A very simple \mbox{\hyperlink{class_a_d_s_r}{ADSR}} envelope class.

To use it, call \mbox{\hyperlink{class_a_d_s_r_a570f2705d7584d8489b01db8e9cf4485}{set\+Sample\+Rate()}} with the current sample rate and give it some parameters with \mbox{\hyperlink{class_a_d_s_r_a621c70a6324e9bfaa3926f3da557ade9}{set\+Parameters()}} then call \mbox{\hyperlink{class_a_d_s_r_a75be2a724854a7f591d0b7960dfeb10f}{get\+Next\+Sample()}} to get the envelope value to be applied to each audio sample or \mbox{\hyperlink{class_a_d_s_r_a883ba7e39999c2c77786cca1b7926694}{apply\+Envelope\+To\+Buffer()}} to apply the envelope to a whole buffer.
