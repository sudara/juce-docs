\doxysubsection{Detailed Description}
Has a callback method that is triggered asynchronously.

This object allows an asynchronous callback function to be triggered, for tasks such as coalescing multiple updates into a single callback later on.

Basically, one or more calls to the \mbox{\hyperlink{class_async_updater_a4eece806c6ba9f591382fed54c5983b2}{trigger\+Async\+Update()}} will result in the message thread calling \mbox{\hyperlink{class_async_updater_a446f3af793131f8772329f1f8b526578}{handle\+Async\+Update()}} as soon as it can.

This will stop any pending updates from happening.

If called after \mbox{\hyperlink{class_async_updater_a4eece806c6ba9f591382fed54c5983b2}{trigger\+Async\+Update()}} and before the \mbox{\hyperlink{class_async_updater_a446f3af793131f8772329f1f8b526578}{handle\+Async\+Update()}} callback happens, this will cancel the \mbox{\hyperlink{class_async_updater_a446f3af793131f8772329f1f8b526578}{handle\+Async\+Update()}} callback.

{\footnotesize\ttfamily \mbox{\hyperlink{png_8h_aa8c59027f9ab2769342f248709d68d17}{void}} Async\+Updater\+::trigger\+Async\+Update (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}

Causes the callback to be triggered at a later time.

This method returns immediately, after which a callback to the \mbox{\hyperlink{class_async_updater_a446f3af793131f8772329f1f8b526578}{handle\+Async\+Update()}} method will be made by the message thread as soon as possible.

If an update callback is already pending but hasn\textquotesingle{}t happened yet, calling this method will have no effect.

It\textquotesingle{}s thread-\/safe to call this method from any thread, BUT beware of calling it from a real-\/time (e.\+g. audio) thread, because it involves posting a message to the system queue, which means it may block (and in general will do on most OSes).
