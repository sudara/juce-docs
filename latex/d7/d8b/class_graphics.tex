\doxysubsection{Detailed Description}
A graphics context, used for drawing a component or image.

When a \mbox{\hyperlink{class_component}{Component}} needs painting, a \mbox{\hyperlink{class_graphics}{Graphics}} context is passed to its \mbox{\hyperlink{class_component_a7cf1862f4af5909ea72827898114a182}{Component\+::paint()}} method, and this you then call methods within this object to actually draw the component\textquotesingle{}s content.

A \mbox{\hyperlink{class_graphics}{Graphics}} can also be created from an image, to allow drawing directly onto that image.

Draws an image, having applied an affine transform to it.

This lets you throw the image around in some wacky ways, rotate it, shear, scale it, etc.

Images are composited using the context\textquotesingle{}s current opacity, so if you don\textquotesingle{}t want it to be drawn semi-\/transparently, be sure to call set\+Opacity (1.\+0f) (or \mbox{\hyperlink{class_graphics_a9a944a0006b7277fda473f0b1b4f028f}{set\+Colour()}} with an opaque colour) before drawing images.

If fill\+Alpha\+Channel\+With\+Current\+Brush is set to true, then the image\textquotesingle{}s RGB channels are ignored and it is filled with the current brush, masked by its alpha channel.

If you want to render only a subsection of an image, use \mbox{\hyperlink{class_image_a0f1e8e0a7e0abf650911df14bbf7e701}{Image\+::get\+Clipped\+Image()}} to create the section that you need.
