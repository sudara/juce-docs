\doxysubsection{Detailed Description}
This wraps a context menu for a specific parameter, as provided by the host.

You can choose to create a standard \mbox{\hyperlink{class_popup_menu}{Popup\+Menu}} to display the host-\/provided options. Alternatively, you can ask the host to display a native menu at a specific location.
