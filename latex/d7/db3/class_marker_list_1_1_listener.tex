\doxysubsection{Detailed Description}
A class for receiving events when changes are made to a \mbox{\hyperlink{class_marker_list}{Marker\+List}}.

You can register a \mbox{\hyperlink{class_marker_list_1_1_listener}{Marker\+List\+::\+Listener}} with a \mbox{\hyperlink{class_marker_list}{Marker\+List}} using the \mbox{\hyperlink{class_marker_list_a69ebc97778614b8abe46134f2bd4fe14}{Marker\+List\+::add\+Listener()}} method, and it will be called when markers are moved, added, or deleted.
