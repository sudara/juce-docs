\doxysubsection{Detailed Description}
A simple wrapper for building a collection of supported accessibility actions and corresponding callbacks for a UI element.

Pass one of these when constructing an {\ttfamily \mbox{\hyperlink{class_accessibility_handler}{Accessibility\+Handler}}} to enable users to interact with a UI element via the supported actions.
