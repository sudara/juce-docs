\doxysubsection{Detailed Description}
A class that can be used in place of a real \mbox{\hyperlink{class_critical_section}{Critical\+Section}} object, but which doesn\textquotesingle{}t perform any locking.

This is currently used by some templated classes, and most compilers should manage to optimise it out of existence.
