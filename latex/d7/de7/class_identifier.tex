\doxysubsection{Detailed Description}
Represents a string identifier, designed for accessing properties by name.

Comparing two \mbox{\hyperlink{class_identifier}{Identifier}} objects is very fast (an O(1) operation), but creating them can be slower than just using a \mbox{\hyperlink{class_string}{String}} directly, so the optimal way to use them is to keep some static \mbox{\hyperlink{class_identifier}{Identifier}} objects for the things you use often.
