\doxysubsection{Detailed Description}
A text string with a set of colour/font settings that are associated with sub-\/ranges of the text.

An attributed string lets you create a string with varied fonts, colours, word-\/wrapping, layout, etc., and draw it using \mbox{\hyperlink{class_attributed_string_a2befaf1fec54e78f844f034986f5f6f8}{Attributed\+String\+::draw()}}.

Invariants\+:
\begin{DoxyItemize}
\item Every character in the string is a member of exactly one attribute.
\item Attributes are sorted such that the range-\/end of attribute \textquotesingle{}i\textquotesingle{} is equal to the range-\/begin of attribute \textquotesingle{}i + 1\textquotesingle{}.
\end{DoxyItemize}
