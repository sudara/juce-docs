\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_steinberg_1_1_update_handler}{Update\+Handler}} implements \mbox{\hyperlink{class_steinberg_1_1_i_update_manager}{IUpdate\+Manager}} and \mbox{\hyperlink{class_steinberg_1_1_i_update_handler}{IUpdate\+Handler}} to handle dependencies between objects to store and forward messages to dependent objects.

This implementation is thread save, so objects can send message, add or remove dependents from different threads. Do do so it uses mutex, so be aware of locking.
