\doxysubsection{Detailed Description}
The class is a base class for objects that manage the loading of a type of component from a \mbox{\hyperlink{class_value_tree}{Value\+Tree}}.

To store and re-\/load a tree of components as a \mbox{\hyperlink{class_value_tree}{Value\+Tree}}, each component type must have a \mbox{\hyperlink{class_component_builder_1_1_type_handler}{Type\+Handler}} to represent it.
