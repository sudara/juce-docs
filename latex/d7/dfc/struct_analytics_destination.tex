\doxysubsection{Detailed Description}
An interface for handling analytics events collected by an \mbox{\hyperlink{class_analytics}{Analytics}} object.

For basic analytics logging you can implement this interface and add your class to an \mbox{\hyperlink{class_analytics}{Analytics}} object.

For more advanced logging you may want to subclass \mbox{\hyperlink{class_threaded_analytics_destination}{Threaded\+Analytics\+Destination}} instead, which is more suitable for interacting with web servers and other time consuming destinations.
