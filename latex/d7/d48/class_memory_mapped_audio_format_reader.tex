\doxysubsection{Detailed Description}
A specialised type of \mbox{\hyperlink{class_audio_format_reader}{Audio\+Format\+Reader}} that uses a \mbox{\hyperlink{class_memory_mapped_file}{Memory\+Mapped\+File}} to read directly from an audio file.

This allows for incredibly fast random-\/access to sample data in the mapped region of the file, but not all audio formats support it -\/ see \mbox{\hyperlink{class_audio_format_afd3b0290541b4dee9ab8e667906f2825}{Audio\+Format\+::create\+Memory\+Mapped\+Reader()}}.

Note that before reading samples from a \mbox{\hyperlink{class_memory_mapped_audio_format_reader}{Memory\+Mapped\+Audio\+Format\+Reader}}, you must first call \mbox{\hyperlink{class_memory_mapped_audio_format_reader_a857ab88ffdc45bb42ca1468ce2b17e35}{map\+Entire\+File()}} or \mbox{\hyperlink{class_memory_mapped_audio_format_reader_aee6db6421fa50c948b724fd362359bf9}{map\+Section\+Of\+File()}} to ensure that the region you want to read has been mapped.
