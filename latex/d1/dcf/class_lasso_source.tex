class Lasso\+Source$<$ Selectable\+Item\+Type $>$}
A class used by the \mbox{\hyperlink{class_lasso_component}{Lasso\+Component}} to manage the things that it selects.

This allows the \mbox{\hyperlink{class_lasso_component}{Lasso\+Component}} to find out which items are within the lasso, and to change the list of selected items.
