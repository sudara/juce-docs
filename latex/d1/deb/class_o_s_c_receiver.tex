\doxysubsection{Detailed Description}
A class for receiving OSC data.

An \mbox{\hyperlink{class_o_s_c_receiver}{OSCReceiver}} object allows you to receive OSC bundles and messages. It can connect to a network port, receive incoming OSC packets from the network via UDP, parse them, and forward the included \mbox{\hyperlink{class_o_s_c_message}{OSCMessage}} and \mbox{\hyperlink{class_o_s_c_bundle}{OSCBundle}} objects to its listeners.
