\doxysubsection{Detailed Description}
A subclass of \mbox{\hyperlink{class_synthesiser_sound}{Synthesiser\+Sound}} that represents a sampled audio clip.

This is a pretty basic sampler, and just attempts to load the whole audio stream into memory.

To use it, create a \mbox{\hyperlink{class_synthesiser}{Synthesiser}}, add some \mbox{\hyperlink{class_sampler_voice}{Sampler\+Voice}} objects to it, then give it some Sampled\+Sound objects to play.
