\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Sample\+Type$>$\newline
class dsp\+::\+Audio\+Block$<$ Sample\+Type $>$}
Minimal and lightweight data-\/structure which contains a list of pointers to channels containing some kind of sample data.

This class doesn\textquotesingle{}t own any of the data which it points to, it\textquotesingle{}s simply a view into data that is owned elsewhere. You can construct one from some raw data that you\textquotesingle{}ve allocated yourself, or give it a \mbox{\hyperlink{class_heap_block}{Heap\+Block}} to use, or give it an \mbox{\hyperlink{class_audio_buffer}{Audio\+Buffer}} which it can refer to, but in all cases the user is responsible for making sure that the data doesn\textquotesingle{}t get deleted while there\textquotesingle{}s still an \mbox{\hyperlink{classdsp_1_1_audio_block}{Audio\+Block}} using it.
