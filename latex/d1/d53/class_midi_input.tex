\doxysubsection{Detailed Description}
Represents a midi input device.

To create one of these, use the static \mbox{\hyperlink{class_midi_input_a5696a5c974be7e2551701dabfcaabe63}{get\+Available\+Devices()}} method to find out what inputs are available, and then use the \mbox{\hyperlink{class_midi_input_a422b09cef1c9d61a1525d28f26c1a7cf}{open\+Device()}} method to try to open one.

Tries to open one of the midi input devices.

This will return a \mbox{\hyperlink{class_midi_input}{Midi\+Input}} object if it manages to open it, you can then call \mbox{\hyperlink{class_midi_input_ae6095bfc60700f95534db37828c72f2b}{start()}} and \mbox{\hyperlink{class_midi_input_a1e97664380c7764a0cdd909c2123f2f0}{stop()}} on this device.

If the device can\textquotesingle{}t be opened, this will return an empty object.
