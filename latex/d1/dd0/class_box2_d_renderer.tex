\doxysubsection{Detailed Description}
A simple implementation of the \mbox{\hyperlink{classb2_draw}{b2\+Draw}} class, used to draw a Box2D world.

To use it, simply create an instance of this class in your paint() method, and call its \mbox{\hyperlink{class_box2_d_renderer_a9dc474dbf89965501a5d92300c46dfa0}{render()}} method.
