\doxysubsection{Detailed Description}
Parses a text-\/based XML document and creates an \mbox{\hyperlink{class_xml_element}{Xml\+Element}} object from it.

The parser will parse DTDs to load external entities but won\textquotesingle{}t check the document for validity against the DTD.

If this is true (the default state), then any text elements that contain only whitespace characters will be ignored during parsing. Sets an input source object to use for parsing documents that reference external entities.

If the document has been created from a file, this probably won\textquotesingle{}t be needed, but if you\textquotesingle{}re parsing some text and there might be a DTD that references external files, you may need to create a custom input source that can retrieve the other files it needs.

The object that is passed-\/in will be deleted automatically when no longer needed.
