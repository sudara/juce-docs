\doxysubsection{Detailed Description}
Android API level 26 or higher only\+: Represents notification channel through which notifications will be sent. Starting from Android API level 26, you should call \mbox{\hyperlink{class_push_notifications_abc63591859e779d21052318d0932a738}{setup\+Channels()}} at the start of your application, before posting any notifications. Then, when sending notifications, assign a channel to each created notification.
