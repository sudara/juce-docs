\doxysubsection{Detailed Description}
\subsubsection*{template$<$size\+\_\+t len, typename Ret, typename... Args$>$\newline
class dsp\+::\+Fixed\+Size\+Function$<$ len, Ret(\+Args...)$>$}
A type similar to {\ttfamily std\+::function} that holds a callable object.

Unlike {\ttfamily std\+::function}, the callable object will always be stored in a buffer of size {\ttfamily len} that is internal to the \mbox{\hyperlink{classdsp_1_1_fixed_size_function}{Fixed\+Size\+Function}} instance. This in turn means that creating a \mbox{\hyperlink{classdsp_1_1_fixed_size_function}{Fixed\+Size\+Function}} instance will never allocate, making Fixed\+Size\+Functions suitable for use in realtime contexts.
