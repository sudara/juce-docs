\doxysubsection{Detailed Description}
One of these is used by a \mbox{\hyperlink{class_table_list_box}{Table\+List\+Box}} as the data model for the table\textquotesingle{}s contents.

The virtual methods that you override in this class take care of drawing the table cells, and reacting to events.
