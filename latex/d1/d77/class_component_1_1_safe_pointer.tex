\doxysubsection{Detailed Description}
\subsubsection*{template$<$class Component\+Type$>$\newline
class Component\+::\+Safe\+Pointer$<$ Component\+Type $>$}
Holds a pointer to some type of \mbox{\hyperlink{class_component}{Component}}, which automatically becomes null if the component is deleted.

If you\textquotesingle{}re using a component which may be deleted by another event that\textquotesingle{}s outside of your control, use a \mbox{\hyperlink{class_component_1_1_safe_pointer}{Safe\+Pointer}} instead of a normal pointer to refer to it, and you can test whether it\textquotesingle{}s null before using it to see if something has deleted it.

The Component\+Type template parameter must be \mbox{\hyperlink{class_component}{Component}}, or some subclass of \mbox{\hyperlink{class_component}{Component}}.

You may also want to use a Weak\+Reference$<$\+Component$>$ object for the same purpose.
