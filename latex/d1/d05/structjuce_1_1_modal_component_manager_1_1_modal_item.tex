Called when the component is in the process of being deleted.

This callback is made from inside the destructor, so be very, very cautious about what you do in here.

In particular, bear in mind that it\textquotesingle{}s the \mbox{\hyperlink{class_component}{Component}} base class\textquotesingle{}s destructor that calls this -\/ so if the object that\textquotesingle{}s being deleted is a subclass of \mbox{\hyperlink{class_component}{Component}}, then the subclass layers of the object will already have been destructed when it gets to this point!
