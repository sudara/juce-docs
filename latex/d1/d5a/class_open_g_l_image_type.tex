\doxysubsection{Detailed Description}
A type of \mbox{\hyperlink{class_image_pixel_data}{Image\+Pixel\+Data}} that stores its image data in an Open\+GL framebuffer, allowing a JUCE \mbox{\hyperlink{class_image}{Image}} object to wrap a framebuffer.

By creating an \mbox{\hyperlink{class_image}{Image}} from an instance of an Open\+GLFrame\+Buffer\+Image, you can then use a \mbox{\hyperlink{class_graphics}{Graphics}} object to draw into the framebuffer using normal JUCE 2D operations.
