Creating one of these automatically sends analytics events to the \mbox{\hyperlink{class_analytics}{Analytics}} singleton when the corresponding button is clicked.

The name and parameters of the analytics event will be populated from the variables supplied here. If clicking changes the button\textquotesingle{}s state then the parameters will have a \{\char`\"{}\+Button\+State\char`\"{}, \char`\"{}\+On\char`\"{}/\char`\"{}\+Off\char`\"{}\} entry added.
