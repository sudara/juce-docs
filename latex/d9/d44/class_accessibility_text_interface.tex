\doxysubsection{Detailed Description}
An abstract interface which represents a UI element that supports a text interface.

A UI element can use this interface to provide extended textual information which cannot be conveyed using just the title, description, and help text properties of \mbox{\hyperlink{class_accessibility_handler}{Accessibility\+Handler}}. This is typically for text that an accessibility client might want to read line-\/by-\/line, or provide text selection and input for.
