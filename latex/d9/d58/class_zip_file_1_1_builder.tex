\doxysubsection{Detailed Description}
Used to create a new zip file.

Create a \mbox{\hyperlink{class_zip_file_1_1_builder}{Zip\+File\+::\+Builder}} object, and call its \mbox{\hyperlink{class_zip_file_1_1_builder_a278af3ba8d510e7895109af905a9c67f}{add\+File()}} method to add some files, then you can write it to a stream with write().
