\doxysubsection{Detailed Description}
A singleton class responsible for sharing content between apps and devices.

You can share text, images, files or an arbitrary data block.

A convenience function to share arbitrary data. The data will be written to a temporary file and then that file will be shared. If you have your data stored on disk already, call \mbox{\hyperlink{class_content_sharer_ad84934f80c996f54e5ed119f360b067a}{share\+Files()}} instead.

Upon completion you will receive a callback with a sharing result. Note\+: Sadly on Android the returned success flag may be wrong as there is no standard way the sharing targets report if the sharing operation succeeded. Also, the optional error message is always empty on Android. \mbox{\Hypertarget{class_content_sharer_ad84934f80c996f54e5ed119f360b067a}\label{class_content_sharer_ad84934f80c996f54e5ed119f360b067a}}

Shares the given files. Each \mbox{\hyperlink{class_u_r_l}{URL}} should be either a full file path or it should point to a resource within the application bundle. For resources on i\+OS it should be something like \char`\"{}content/image.\+png\char`\"{} if you want to specify a file from application bundle located in \char`\"{}content\char`\"{} directory. On Android you should specify only a filename, without an extension.

Upon completion you will receive a callback with a sharing result. Note\+: Sadly on Android the returned success flag may be wrong as there is no standard way the sharing targets report if the sharing operation succeeded.
