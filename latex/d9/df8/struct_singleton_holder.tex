struct Singleton\+Holder$<$ Type, Mutex\+Type, only\+Create\+Once\+Per\+Run $>$}
Used by the JUCE\+\_\+\+DECLARE\+\_\+\+SINGLETON macros to manage a static pointer to a singleton instance.

You generally won\textquotesingle{}t use this directly, but see the macros JUCE\+\_\+\+DECLARE\+\_\+\+SINGLETON, JUCE\+\_\+\+DECLARE\+\_\+\+SINGLETON\+\_\+\+SINGLETHREADED, JUCE\+\_\+\+DECLARE\+\_\+\+SINGLETON\+\_\+\+SINGLETHREADED\+\_\+\+MINIMAL, and JUCE\+\_\+\+IMPLEMENT\+\_\+\+SINGLETON for how it is intended to be used.
