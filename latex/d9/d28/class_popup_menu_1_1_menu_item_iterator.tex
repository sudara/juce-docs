\doxysubsection{Detailed Description}
Allows you to iterate through the items in a pop-\/up menu, and examine their properties.

To use this, just create one and repeatedly call its \mbox{\hyperlink{class_popup_menu_1_1_menu_item_iterator_a1f3642d772ba69f95d0b90db80cc3d71}{next()}} method. When this returns true, all the member variables of the iterator are filled-\/out with information describing the menu item. When it returns false, the end of the list has been reached.
