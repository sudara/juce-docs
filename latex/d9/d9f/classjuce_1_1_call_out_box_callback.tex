Called to indicate that a modal component has been dismissed.

You can register a callback using \mbox{\hyperlink{class_component_a0e7be8c456a663615eb389ecd10b2c04}{Component\+::enter\+Modal\+State()}} or \mbox{\hyperlink{class_modal_component_manager_a2ce962253cf2c8e42d5575ad2a41d888}{Modal\+Component\+Manager\+::attach\+Callback()}}.

The return\+Value parameter is the value that was passed to \mbox{\hyperlink{class_component_a686e2440370ff4a97c7a7851907be650}{Component\+::exit\+Modal\+State()}} when the component was dismissed.

The callback object will be deleted shortly after this method is called.
