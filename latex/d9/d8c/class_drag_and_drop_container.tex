\doxysubsection{Detailed Description}
Enables drag-\/and-\/drop behaviour for a component and all its sub-\/components.

For a component to be able to make or receive drag-\/and-\/drop events, one of its parent components must derive from this class. It\textquotesingle{}s probably best for the top-\/level component to implement it.

Then to start a drag operation, any sub-\/component can just call the \mbox{\hyperlink{class_drag_and_drop_container_afd1be36dcfffa57d879de619dab55497}{start\+Dragging()}} method, and this object will take over, tracking the mouse and sending appropriate callbacks to any child components derived from \mbox{\hyperlink{class_drag_and_drop_target}{Drag\+And\+Drop\+Target}} which the mouse moves over.

Note\+: If all that you need to do is to respond to files being drag-\/and-\/dropped from the operating system onto your component, you don\textquotesingle{}t need any of these classes\+: you can do this simply by overriding \mbox{\hyperlink{class_file_drag_and_drop_target_adc7848885ab2d9380f242c6445b019d4}{File\+Drag\+And\+Drop\+Target\+::files\+Dropped()}}.
