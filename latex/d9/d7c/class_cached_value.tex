\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Type$>$\newline
class Cached\+Value$<$ Type $>$}
This class acts as a typed wrapper around a property inside a \mbox{\hyperlink{class_value_tree}{Value\+Tree}}.

A \mbox{\hyperlink{class_cached_value}{Cached\+Value}} provides an easy way to read and write a \mbox{\hyperlink{class_value_tree}{Value\+Tree}} property with a chosen type. So for example a Cached\+Value$<$int$>$ allows you to read or write the property as an int, and a Cached\+Value$<$\+String$>$ lets you work with it as a \mbox{\hyperlink{class_string}{String}}.

It also allows efficient access to the value, by caching a copy of it in the type that is being used.

You can give the \mbox{\hyperlink{class_cached_value}{Cached\+Value}} an optional \mbox{\hyperlink{class_undo_manager}{Undo\+Manager}} which it will use when writing to the underlying \mbox{\hyperlink{class_value_tree}{Value\+Tree}}.

If the property inside the \mbox{\hyperlink{class_value_tree}{Value\+Tree}} is missing, the \mbox{\hyperlink{class_cached_value}{Cached\+Value}} will automatically return an optional default value, which can be specified when initialising the \mbox{\hyperlink{class_cached_value}{Cached\+Value}}.

To create one, you can either use the constructor to attach the \mbox{\hyperlink{class_cached_value}{Cached\+Value}} to a \mbox{\hyperlink{class_value_tree}{Value\+Tree}}, or can create an uninitialised \mbox{\hyperlink{class_cached_value}{Cached\+Value}} with its default constructor and then attach it later with the \mbox{\hyperlink{group__juce__data__structures-values_ga22c029d41ad7c38c01d91e8b77bdcae6}{refer\+To()}} methods.

Common types like \mbox{\hyperlink{class_string}{String}}, int, double which can be easily converted to a var should work out-\/of-\/the-\/box, but if you want to use more complex custom types, you may need to implement some template specialisations of \mbox{\hyperlink{struct_variant_converter}{Variant\+Converter}} which this class uses to convert between the type and the \mbox{\hyperlink{class_value_tree}{Value\+Tree}}\textquotesingle{}s internal var.
