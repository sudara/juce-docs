\doxysubsection{Detailed Description}
A class for receiving events from a \mbox{\hyperlink{class_label}{Label}}.

You can register a \mbox{\hyperlink{class_label_1_1_listener}{Label\+::\+Listener}} with a \mbox{\hyperlink{class_label}{Label}} using the \mbox{\hyperlink{class_label_a05dc02ef40bf0244c928bff8c7cf904e}{Label\+::add\+Listener()}} method, and it will be called when the text of the label changes, either because of a call to \mbox{\hyperlink{class_label_a3f0ca22cb63e924d3db23da48c210790}{Label\+::set\+Text()}} or by the user editing the text (if the label is editable).
