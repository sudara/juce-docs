\doxysubsection{Detailed Description}
Base class for objects that can be used to automatically position a component according to some kind of algorithm.

The component class simply holds onto a reference to a \mbox{\hyperlink{class_component_1_1_positioner}{Positioner}}, but doesn\textquotesingle{}t actually do anything with it -\/ all the functionality must be implemented by the positioner itself (e.\+g. it might choose to watch some kind of value and move the component when the value changes).
