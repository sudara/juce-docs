\doxysubsection{Detailed Description}
A factory object which can create \mbox{\hyperlink{class_toolbar_item_component}{Toolbar\+Item\+Component}} objects.

A subclass of \mbox{\hyperlink{class_toolbar_item_factory}{Toolbar\+Item\+Factory}} publishes a set of types of toolbar item that it can create.

Each type of item is identified by a unique ID, and multiple instances of an item type can exist at once (even on the same toolbar, e.\+g. spacers or separator bars).

Must return a list of the IDs for all the item types that this factory can create.

The ids should be added to the array that is passed-\/in.

An item ID can be any integer you choose, except for 0, which is considered a null ID, and the predefined IDs in the Special\+Item\+Ids enum.

You should also add the built-\/in types (separator\+Bar\+Id, spacer\+Id and flexible\+Spacer\+Id) to this list if you want your toolbar to be able to contain those items.

The list returned here is used by the \mbox{\hyperlink{class_toolbar_item_palette}{Toolbar\+Item\+Palette}} class to obtain its list of available items, and their order on the palette will reflect the order in which they appear on this list.
