\doxysubsection{Detailed Description}
Holds a collection of Universal MIDI \mbox{\hyperlink{classuniversal__midi__packets_1_1_packets}{Packets}}.

Unlike \mbox{\hyperlink{class_midi_buffer}{Midi\+Buffer}}, this collection does not store any additional information (e.\+g. timestamps) alongside the raw messages.

If timestamps are required, these can be added to the container in UMP format, as Jitter Reduction Utility messages.
