Called when the application starts.

This will be called once to let the application do whatever initialisation it needs, create its windows, etc.

After the method returns, the normal event-\/dispatch loop will be run, until the \mbox{\hyperlink{class_j_u_c_e_application_base_a542e212605f0aa693f687d8798998615}{quit()}} method is called, at which point the \mbox{\hyperlink{classjuce_1_1_standalone_filter_app_a22603005bb0ca058a1155cbf2973c985}{shutdown()}} method will be called to let the application clear up anything it needs to delete.

If during the \mbox{\hyperlink{classjuce_1_1_standalone_filter_app_a053fa7bbcaffeed0bc297bc13ef74235}{initialise()}} method, the application decides not to start-\/up after all, it can just call the \mbox{\hyperlink{class_j_u_c_e_application_base_a542e212605f0aa693f687d8798998615}{quit()}} method and the event loop won\textquotesingle{}t be run.

Checks whether multiple instances of the app are allowed.

If your application class returns true for this, more than one instance is permitted to run (except on the Mac where this isn\textquotesingle{}t possible).

If it\textquotesingle{}s false, the second instance won\textquotesingle{}t start, but you will still get a callback to \mbox{\hyperlink{classjuce_1_1_standalone_filter_app_a8e6b86e89d681d91d426a16de3861e12}{another\+Instance\+Started()}} to tell you about this -\/ which gives you a chance to react to what the user was trying to do.

Called when the operating system is trying to close the application.

The default implementation of this method is to call \mbox{\hyperlink{class_j_u_c_e_application_base_a542e212605f0aa693f687d8798998615}{quit()}}, but it may be overloaded to ignore the request or do some other special behaviour instead. For example, you might want to offer the user the chance to save their changes before quitting, and give them the chance to cancel.

If you want to send a quit signal to your app, this is the correct method to call, because it means that requests that come from the system get handled in the same way as those from your own application code. So e.\+g. you\textquotesingle{}d call this method from a \char`\"{}quit\char`\"{} item on a menu bar.
