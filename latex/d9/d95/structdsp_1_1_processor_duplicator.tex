\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Mono\+Processor\+Type, typename State\+Type$>$\newline
struct dsp\+::\+Processor\+Duplicator$<$ Mono\+Processor\+Type, State\+Type $>$}
Converts a mono processor class into a multi-\/channel version by duplicating it and applying multichannel buffers across an array of instances.

When the prepare method is called, it uses the specified number of channels to instantiate the appropriate number of instances, which it then uses in its \mbox{\hyperlink{structdsp_1_1_processor_duplicator_a2dbc05482b6db273eac54fd097d021f4}{process()}} method.
