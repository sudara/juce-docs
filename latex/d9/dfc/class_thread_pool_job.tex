\doxysubsection{Detailed Description}
A task that is executed by a \mbox{\hyperlink{class_thread_pool}{Thread\+Pool}} object.

A \mbox{\hyperlink{class_thread_pool}{Thread\+Pool}} keeps a list of \mbox{\hyperlink{class_thread_pool_job}{Thread\+Pool\+Job}} objects which are executed by its threads.

The \mbox{\hyperlink{class_thread_pool_job_aec49c01680e563257c79088dc87bc0d9}{run\+Job()}} method needs to be implemented to do the task, and if the code that does the work takes a significant time to run, it must keep checking the \mbox{\hyperlink{class_thread_pool_job_a441a94cba7ec96242fd7268371d9c5ea}{should\+Exit()}} method to see if something is trying to interrupt the job. If \mbox{\hyperlink{class_thread_pool_job_a441a94cba7ec96242fd7268371d9c5ea}{should\+Exit()}} returns true, the \mbox{\hyperlink{class_thread_pool_job_aec49c01680e563257c79088dc87bc0d9}{run\+Job()}} method must return immediately.

Performs the actual work that this job needs to do.

Your subclass must implement this method, in which is does its work.

If the code in this method takes a significant time to run, it must repeatedly check the \mbox{\hyperlink{class_thread_pool_job_a441a94cba7ec96242fd7268371d9c5ea}{should\+Exit()}} method to see if something is trying to interrupt the job. If \mbox{\hyperlink{class_thread_pool_job_a441a94cba7ec96242fd7268371d9c5ea}{should\+Exit()}} ever returns true, the \mbox{\hyperlink{class_thread_pool_job_aec49c01680e563257c79088dc87bc0d9}{run\+Job()}} method must return immediately.

If this method returns job\+Has\+Finished, then the job will be removed from the pool immediately. If it returns job\+Needs\+Running\+Again, then the job will be left in the pool and will get a chance to run again as soon as a thread is free.
