{\footnotesize\ttfamily enum \mbox{\hyperlink{class_label_a41756012394513222e1323bb432fa701}{Label\+::\+Colour\+Ids}}}

A set of colour IDs to use to change the colour of various aspects of the label.

These constants can be used either via the \mbox{\hyperlink{class_component_aad68f45fb458d0cac645c0910193142d}{Component\+::set\+Colour()}}, or \mbox{\hyperlink{class_look_and_feel_a167a1e914771f32e433a0d45aaba45e3}{Look\+And\+Feel\+::set\+Colour()}} methods.

Note that you can also use the constants from \mbox{\hyperlink{class_text_editor_aa805220923c93cd808a92fff0e3fb71d}{Text\+Editor\+::\+Colour\+Ids}} to change the colour of the text editor that is opened when a label is editable.

Components can override this method to draw their content.

The \mbox{\hyperlink{class_label_a5afd583115f3a166cdd586d735d4d7a0}{paint()}} method gets called when a region of a component needs redrawing, either because the component\textquotesingle{}s \mbox{\hyperlink{class_component_af3ab3eabec93dd60faf983370f3a1206}{repaint()}} method has been called, or because something has happened on the screen that means a section of a window needs to be redrawn.

Any child components will draw themselves over whatever this method draws. If you need to paint over the top of your child components, you can also implement the \mbox{\hyperlink{class_component_aa15dc77c06b5a0a4192090b4a323f79c}{paint\+Over\+Children()}} method to do this.

If you want to cause a component to redraw itself, this is done asynchronously -\/ calling the \mbox{\hyperlink{class_component_af3ab3eabec93dd60faf983370f3a1206}{repaint()}} method marks a region of the component as \char`\"{}dirty\char`\"{}, and the \mbox{\hyperlink{class_label_a5afd583115f3a166cdd586d735d4d7a0}{paint()}} method will automatically be called sometime later, by the message thread, to paint any bits that need refreshing. In JUCE (and almost all modern UI frameworks), you never redraw something synchronously.

You should never need to call this method directly -\/ to take a snapshot of the component you could use \mbox{\hyperlink{class_component_a275e46e5a8147f89c142c91ba5c793a5}{create\+Component\+Snapshot()}} or \mbox{\hyperlink{class_component_a2c960bd5ec5ac76d18e82b6883096714}{paint\+Entire\+Component()}}.
