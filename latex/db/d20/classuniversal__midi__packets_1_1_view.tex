\doxysubsection{Detailed Description}
Points to a single Universal MIDI \mbox{\hyperlink{classuniversal__midi__packets_1_1_packet}{Packet}}.

The packet must be well-\/formed for member functions to work correctly.

Specifically, the constructor argument must be the beginning of a region of uint32\+\_\+t that contains at least {\ttfamily get\+Num\+Words\+For\+Message\+Type($\ast$ddata)} items, where {\ttfamily data} is the constructor argument.

NOTE\+: Instances of this class do not own the memory that they point to! If you need to store a packet pointed-\/to by a \mbox{\hyperlink{classuniversal__midi__packets_1_1_view}{View}} for later use, copy the view contents to a \mbox{\hyperlink{classuniversal__midi__packets_1_1_packets}{Packets}} collection, or use the Utils\+::\+PacketX types.
