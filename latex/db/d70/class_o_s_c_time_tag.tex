\doxysubsection{Detailed Description}
An OSC time tag.

OSC time tags are part of \mbox{\hyperlink{class_o_s_c_bundle}{OSCBundle}} objects.

In accordance with the OSC 1.\+0 specification, the internal timestamp stored in \mbox{\hyperlink{class_o_s_c_time_tag}{OSCTime\+Tag}} uses the same binary format as NTP timestamps. The representation is by a 64 bit fixed point number. The first 32 bits specify the number of seconds since midnight on January 1, 1900, and the last 32 bits specify fractional parts of a second to a precision of about 200 picoseconds.

The time tag value consisting of 63 zero bits followed by a one in the least significant bit is a special case meaning \char`\"{}immediately\char`\"{}.

For a more user-\/friendly time format, convert \mbox{\hyperlink{class_o_s_c_time_tag}{OSCTime\+Tag}} to a juce\+::\+Time object using \mbox{\hyperlink{class_o_s_c_time_tag_ac433f096735c43fe1214eed57315fc8a}{to\+Time()}}.
