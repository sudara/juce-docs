\doxysubsection{Detailed Description}
Animates a set of components, moving them to a new position and/or fading their alpha levels.

To animate a component, create a \mbox{\hyperlink{class_component_animator}{Component\+Animator}} instance or (preferably) use the global animator object provided by \mbox{\hyperlink{class_desktop_ad52c8dba1af50279acdfe5c7ff9ad9c0}{Desktop\+::get\+Animator()}}, and call its \mbox{\hyperlink{class_component_animator_ac4606415aa267cdb0be7e8f17abd8435}{animate\+Component()}} method to commence the movement.

If you\textquotesingle{}re using your own \mbox{\hyperlink{class_component_animator}{Component\+Animator}} instance, you\textquotesingle{}ll need to make sure it isn\textquotesingle{}t deleted before it finishes moving the components, or they\textquotesingle{}ll be abandoned before reaching their destinations.

It\textquotesingle{}s ok to delete components while they\textquotesingle{}re being animated -\/ the animator will detect this and safely stop using them.

The class is a \mbox{\hyperlink{class_change_broadcaster}{Change\+Broadcaster}} and sends a notification when any components start or finish being animated.

Starts a component moving from its current position to a specified position.

If the component is already in the middle of an animation, that will be abandoned, and a new animation will begin, moving the component from its current location.

The start and end speed parameters let you apply some acceleration to the component\textquotesingle{}s movement.

{\em use\+Proxy\+Component} & if true, this means the component should be replaced by an internally managed temporary component which is a snapshot of the original component. This avoids the component having to paint itself as it moves, so may be more efficient. This option also allows you to delete the original component immediately after starting the animation, because the animation can proceed without it. If you use a proxy, the original component will be made invisible by this call, and then will become visible again at the end of the animation. It\textquotesingle{}ll also mean that the proxy component will be temporarily added to the component\textquotesingle{}s parent, so avoid it if this might confuse the parent component, or if there\textquotesingle{}s a chance the parent might decide to delete its children. \\
\hline
{\em start\+Speed} & a value to indicate the relative start speed of the animation. If this is 0, the component will start by accelerating from rest; higher values mean that it will have an initial speed greater than zero. If the value is greater than 1, it will decelerate towards the middle of its journey. To move the component at a constant rate for its entire animation, set both the start and end speeds to 1.\+0 \\
\hline

If this is 0, the component will decelerate to a standstill at its final position; higher values mean the component will still be moving when it stops.
