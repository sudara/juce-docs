Generates the content of a key-\/file which can be sent to a user\textquotesingle{}s machine to unlock a product.

The returned value is a block of text containing an RSA-\/encoded block, followed by some human-\/readable details. If you pass this block of text to \mbox{\hyperlink{class_online_unlock_status_ac0763c3b120f1b173c46dc5611112da3}{Online\+Unlock\+Status\+::apply\+Key\+File()}}, it will decrypt it, and if the key matches and the machine numbers match, it will unlock that machine.

Typically the way you\textquotesingle{}d use this on a server would be to build a small executable that simply calls this method and prints the result, so that the webserver can use this as a reply to the product\textquotesingle{}s auto-\/registration mechanism. The \mbox{\hyperlink{class_key_generation_a608c4ffe01796eba24a5ab563b5d488e}{key\+Generation\+App\+Main()}} function is an example of how to build such a function.
