\doxysubsection{Detailed Description}
Holds a list of command-\/line arguments, and provides useful methods for searching and operating on them.

You can create an \mbox{\hyperlink{struct_argument_list}{Argument\+List}} manually, or give it some argv/argc values from a main() function to parse.
