\doxysubsection{Detailed Description}
Adds reference-\/counting to an object.

This is effectively a version of the \mbox{\hyperlink{class_reference_counted_object}{Reference\+Counted\+Object}} class, but which uses a non-\/atomic counter, and so is not thread-\/safe (but which will be more efficient). For more details on how to use it, see the \mbox{\hyperlink{class_reference_counted_object}{Reference\+Counted\+Object}} class notes.
