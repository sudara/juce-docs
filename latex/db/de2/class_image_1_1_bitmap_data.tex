\doxysubsection{Detailed Description}
Retrieves a section of an image as raw pixel data, so it can be read or written to.

You should only use this class as a last resort -\/ messing about with the internals of an image is only recommended for people who really know what they\textquotesingle{}re doing!

A \mbox{\hyperlink{class_image_1_1_bitmap_data}{Bitmap\+Data}} object should be used as a temporary, stack-\/based object. Don\textquotesingle{}t keep one hanging around while the image is being used elsewhere.

Depending on the way the image class is implemented, this may create a temporary buffer which is copied back to the image when the object is deleted, or it may just get a pointer directly into the image\textquotesingle{}s raw data.

You can use the stride and data values in this class directly, but don\textquotesingle{}t alter them! The actual format of the pixel data depends on the image\textquotesingle{}s format -\/ see \mbox{\hyperlink{class_image_a349e7725b438475e29dee3d394e0bd23}{Image\+::get\+Format()}}, and the \mbox{\hyperlink{class_pixel_r_g_b}{Pixel\+RGB}}, \mbox{\hyperlink{class_pixel_a_r_g_b}{Pixel\+ARGB}} and \mbox{\hyperlink{class_pixel_alpha}{Pixel\+Alpha}} classes for more info.
