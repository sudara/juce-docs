\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_value_tree_1_1_listener}{Listener}} class for events that happen to a \mbox{\hyperlink{class_value_tree}{Value\+Tree}}.

To get events from a \mbox{\hyperlink{class_value_tree}{Value\+Tree}}, make your class implement this interface, and use \mbox{\hyperlink{class_value_tree_a0a0d82471cb1119fc1a7b7018e3af394}{Value\+Tree\+::add\+Listener()}} and \mbox{\hyperlink{class_value_tree_ac8930aa94cb6e3714ef9e7449d08e5f6}{Value\+Tree\+::remove\+Listener()}} to register it.
