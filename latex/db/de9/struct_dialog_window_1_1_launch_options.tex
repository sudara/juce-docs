\doxysubsection{Detailed Description}
This class defines a collection of settings to be used to open a \mbox{\hyperlink{class_dialog_window}{Dialog\+Window}}.

The easiest way to open a \mbox{\hyperlink{class_dialog_window}{Dialog\+Window}} is to create yourself a \mbox{\hyperlink{struct_dialog_window_1_1_launch_options}{Launch\+Options}} structure, initialise its fields with the appropriate details, and then call its \mbox{\hyperlink{struct_dialog_window_1_1_launch_options_a050f81198bd25651d16cabf85bb77770}{launch\+Async()}} method to launch the dialog.

Launches a new modal dialog window. This will create a dialog based on the settings in this structure, launch it modally, and return immediately. The window that is returned will be automatically deleted when the modal state is terminated.

When the dialog\textquotesingle{}s close button is clicked, it\textquotesingle{}ll automatically terminate its modal state, but you can also do this programmatically by calling exit\+Modal\+State (return\+Value) on the \mbox{\hyperlink{class_dialog_window}{Dialog\+Window}}.
