\doxysubsection{Detailed Description}
Base class for accessible Components.

This class wraps a \mbox{\hyperlink{class_component}{Component}} and provides methods that allow an accessibility client, such as Voice\+Over on mac\+OS, or Narrator on Windows, to control it.

It handles hierarchical navigation, properties, state, and various interfaces.

Constructor.

This will create a \mbox{\hyperlink{class_accessibility_handler}{Accessibility\+Handler}} which wraps the provided \mbox{\hyperlink{class_component}{Component}} and makes it visible to accessibility clients. You must also specify a role for the UI element from the {\ttfamily Accessibility\+Role} list which best describes it.

To enable users to interact with the UI element you should provide the set of supported actions and their associated callbacks via the {\ttfamily accessibility\+Actions} parameter.

For UI elements that support more complex interaction the value, text, table, and cell interfaces should be implemented as required and passed as the final argument of this constructor.
