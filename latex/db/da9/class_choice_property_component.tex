\doxysubsection{Detailed Description}
A \mbox{\hyperlink{class_property_component}{Property\+Component}} that shows its value as a combo box.

This type of property component contains a list of options and has a combo box to choose one.

Your subclass\textquotesingle{}s constructor must add some strings to the choices \mbox{\hyperlink{class_string_array}{String\+Array}} and these are shown in the list.

The \mbox{\hyperlink{class_choice_property_component_aa70e4010548752d17ba01abb4f7ca623}{get\+Index()}} method will be called to find out which option is the currently selected one. If you call \mbox{\hyperlink{class_choice_property_component_a58f78f0a2820920be803a0513fbe961d}{refresh()}} it will call \mbox{\hyperlink{class_choice_property_component_aa70e4010548752d17ba01abb4f7ca623}{get\+Index()}} to check whether the value has changed, and will update the combo box if needed.

If the user selects a different item from the list, \mbox{\hyperlink{class_choice_property_component_a0e2ae06d2f6b5e9b8dae53c09da63397}{set\+Index()}} will be called to let your class process this.

Updates the property component if the item it refers to has changed.

A subclass must implement this method, and other objects may call it to force it to refresh itself.

The subclass should be economical in the amount of work is done, so for example it should check whether it really needs to do a repaint rather than just doing one every time this method is called, as it may be called when the value being displayed hasn\textquotesingle{}t actually changed.
