\doxysubsection{Detailed Description}
This class is used when references to images need to be stored in Value\+Trees.

An instance of an \mbox{\hyperlink{class_component_builder_1_1_image_provider}{Image\+Provider}} provides a mechanism for converting an \mbox{\hyperlink{class_image}{Image}} to/from a reference, which may be a file, \mbox{\hyperlink{class_u_r_l}{URL}}, ID string, or whatever system is appropriate in your app.

When you\textquotesingle{}re loading components from a \mbox{\hyperlink{class_value_tree}{Value\+Tree}} that may need a way of loading images, you should call \mbox{\hyperlink{class_component_builder_a4eac18667dc404ad636bb72d8e836667}{Component\+Builder\+::set\+Image\+Provider()}} to supply a suitable provider before trying to load the component.
