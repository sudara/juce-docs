\doxysubsection{Detailed Description}
Provides an {\ttfamily operator()} which can create an input handler for a given \mbox{\hyperlink{class_midi_input}{Midi\+Input}}.

All handler classes should have a similar \mbox{\hyperlink{classuniversal__midi__packets_1_1_u32_to_u_m_p_handler_1_1_factory}{Factory}} to facilitate creation of handlers in generic contexts.
