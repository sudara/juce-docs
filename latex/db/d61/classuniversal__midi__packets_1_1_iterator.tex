\doxysubsection{Detailed Description}
Enables iteration over a collection of Universal MIDI \mbox{\hyperlink{classuniversal__midi__packets_1_1_packets}{Packets}} stored as a contiguous range of 32-\/bit words.

This iterator is used by \mbox{\hyperlink{classuniversal__midi__packets_1_1_packets}{Packets}} to allow access to the messages that it contains.
