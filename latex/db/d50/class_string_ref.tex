\doxysubsection{Detailed Description}
A simple class for holding temporary references to a string literal or \mbox{\hyperlink{class_string}{String}}.

Unlike a real \mbox{\hyperlink{class_string}{String}} object, the \mbox{\hyperlink{class_string_ref}{String\+Ref}} does not allocate any memory or take ownership of the strings you give to it -\/ it simply holds a reference to a string that has been allocated elsewhere. The main purpose of the class is to be used instead of a const \mbox{\hyperlink{class_string}{String}}\& as the type of function arguments where the caller may pass either a string literal or a \mbox{\hyperlink{class_string}{String}} object. This means that when the called uses a string literal, there\textquotesingle{}s no need for an temporary \mbox{\hyperlink{class_string}{String}} object to be allocated, and this cuts down overheads substantially.

Because the class is simply a wrapper around a pointer, you should always pass it by value, not by reference.

Bear in mind that there are still many cases where it\textquotesingle{}s better to use an argument which is a const \mbox{\hyperlink{class_string}{String}}\&. For example if the function stores the string or needs to internally create a \mbox{\hyperlink{class_string}{String}} from the argument, then it\textquotesingle{}s better for the original argument to already be a \mbox{\hyperlink{class_string}{String}}.
