\doxysubsection{Detailed Description}
A lowest-\/common-\/denominator implementation of \mbox{\hyperlink{class_low_level_graphics_context}{Low\+Level\+Graphics\+Context}} that does all its rendering in memory.

User code is not supposed to create instances of this class directly -\/ do all your rendering via the \mbox{\hyperlink{class_graphics}{Graphics}} class instead.
