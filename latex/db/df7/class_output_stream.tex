\doxysubsection{Detailed Description}
The base class for streams that write data to some kind of destination.

Input and output streams are used throughout the library -\/ subclasses can override some or all of the virtual functions to implement their behaviour.

Writes a condensed binary encoding of a 32-\/bit integer.

If you\textquotesingle{}re storing a lot of integers which are unlikely to have very large values, this can save a lot of space, because values under 0xff will only take up 2 bytes, under 0xffff only 3 bytes, etc.

The format used is\+: number of significant bytes + up to 4 bytes in little-\/endian order.

It can either write the text as UTF-\/8 or UTF-\/16, and can also add the UTF-\/16 byte-\/order-\/mark bytes (0xff, 0xfe) to indicate the endianness (these should only be used at the start of a file).

If you pass \char`\"{}\textbackslash{}\textbackslash{}n\char`\"{} or \char`\"{}\textbackslash{}\textbackslash{}r\textbackslash{}\textbackslash{}n\char`\"{} then this function will replace any existing line feeds.
