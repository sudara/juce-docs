Opens a file and maps it to an area of virtual memory.

The file should already exist, and should already be the size that you want to work with when you call this. If the file is resized after being opened, the behaviour is undefined.

If the file exists and the operation succeeds, the \mbox{\hyperlink{class_memory_mapped_file_a0855b3adec7d0b893a3e42c542b229a4}{get\+Data()}} and \mbox{\hyperlink{class_memory_mapped_file_a98d9f4380b1343520779570fbe5bf2b0}{get\+Size()}} methods will return the location and size of the data that can be read or written. Note that the entire file is not read into memory immediately -\/ the OS simply creates a virtual mapping, which will lazily pull the data into memory when blocks are accessed.

If the file can\textquotesingle{}t be opened for some reason, the \mbox{\hyperlink{class_memory_mapped_file_a0855b3adec7d0b893a3e42c542b229a4}{get\+Data()}} method will return a null pointer.

If exclusive is false then other apps can also open the same memory mapped file and use this mapping as an effective way of communicating. If exclusive is true then the mapped file will be opened exclusively -\/ preventing other apps to access the file which may improve the performance of accessing the file. \mbox{\Hypertarget{class_memory_mapped_file_adb4bb69bc651e2039fb5fe74b653a1d8}\label{class_memory_mapped_file_adb4bb69bc651e2039fb5fe74b653a1d8}}

Opens a section of a file and maps it to an area of virtual memory.

The file should already exist, and should already be the size that you want to work with when you call this. If the file is resized after being opened, the behaviour is undefined.

If the file exists and the operation succeeds, the \mbox{\hyperlink{class_memory_mapped_file_a0855b3adec7d0b893a3e42c542b229a4}{get\+Data()}} and \mbox{\hyperlink{class_memory_mapped_file_a98d9f4380b1343520779570fbe5bf2b0}{get\+Size()}} methods will return the location and size of the data that can be read or written. Note that the entire file is not read into memory immediately -\/ the OS simply creates a virtual mapping, which will lazily pull the data into memory when blocks are accessed.

If the file can\textquotesingle{}t be opened for some reason, the \mbox{\hyperlink{class_memory_mapped_file_a0855b3adec7d0b893a3e42c542b229a4}{get\+Data()}} method will return a null pointer.

NOTE\+: The start of the actual range used may be rounded-\/down to a multiple of the OS\textquotesingle{}s page-\/size, so do not assume that the mapped memory will begin at exactly the position you requested -\/ always use \mbox{\hyperlink{class_memory_mapped_file_af02af3af1c9c9b1140de4e31c05c375c}{get\+Range()}} to check the actual range that is being used. \mbox{\Hypertarget{class_memory_mapped_file_a50f37ae28ddf9c0e57823c0b75743878}\label{class_memory_mapped_file_a50f37ae28ddf9c0e57823c0b75743878}}
