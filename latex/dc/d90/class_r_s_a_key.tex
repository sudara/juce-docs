\doxysubsection{Detailed Description}
RSA public/private key-\/pair encryption class.

An object of this type makes up one half of a public/private RSA key pair. Use the \mbox{\hyperlink{class_r_s_a_key_ae27bfa583a730332de763fe5ce3b71cf}{create\+Key\+Pair()}} method to create a matching pair for encoding/decoding.

Encodes or decodes a value.

Call this on the public key object to encode some data, then use the matching private key object to decode it.

Returns false if the operation couldn\textquotesingle{}t be completed, e.\+g. if this key hasn\textquotesingle{}t been initialised correctly.

NOTE\+: This method dumbly applies this key to this data. If you encode some data and then try to decode it with a key that doesn\textquotesingle{}t match, this method will still happily do its job and return true, but the result won\textquotesingle{}t be what you were expecting. Creates a public/private key-\/pair.

Each key will perform one-\/way encryption that can only be reversed by using the other key.

The num\+Bits parameter specifies the size of key, e.\+g. 128, 256, 512 bit. Bigger sizes are more secure, but this method will take longer to execute.

The random\+Seeds parameter lets you optionally pass it a set of values with which to seed the random number generation, improving the security of the keys generated. If you supply these, make sure you provide more than 2 values, and the more your provide, the better the security. \mbox{\Hypertarget{class_r_s_a_key_a898ca7521354c74d24ace4689b983f98}\label{class_r_s_a_key_a898ca7521354c74d24ace4689b983f98}}
