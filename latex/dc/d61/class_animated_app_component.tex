\doxysubsection{Detailed Description}
A base class for writing simple one-\/page graphical apps.

A subclass can inherit from this and implement just a few methods such as \mbox{\hyperlink{class_component_a7cf1862f4af5909ea72827898114a182}{paint()}} and mouse-\/handling. The base class provides some simple abstractions to take care of continuously repainting itself.
