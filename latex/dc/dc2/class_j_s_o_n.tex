\doxysubsection{Detailed Description}
Contains static methods for converting JSON-\/formatted text to and from var objects.

The var class is structurally compatible with JSON-\/formatted data, so these functions allow you to parse \mbox{\hyperlink{class_j_s_o_n}{JSON}} into a var object, and to convert a var object to JSON-\/formatted text.
