\doxysubsection{Detailed Description}
Represents the state of an accessible UI element.

An instance of this class is returned by {\ttfamily \mbox{\hyperlink{class_accessibility_handler_abf086024d6f96f1d99d7675cbe3577d7}{Accessibility\+Handler\+::get\+Current\+State()}}} to convey its current state to an accessibility client.
