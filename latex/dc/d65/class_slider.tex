\doxysubsection{Detailed Description}
A slider control for changing a value.

The slider can be horizontal, vertical, or rotary, and can optionally have a text-\/box inside it to show an editable display of the current value.

To use it, create a \mbox{\hyperlink{class_slider}{Slider}} object and use the \mbox{\hyperlink{class_slider_a6b6917dd3753c7552778977733f0b9ef}{set\+Slider\+Style()}} method to set up the type you want. To set up the text-\/entry box, use \mbox{\hyperlink{class_slider_a5bc748a21e72fe14153bc9fe5ac03e77}{set\+Text\+Box\+Style()}}.

To define the values that it can be set to, see the \mbox{\hyperlink{class_slider_a9fbc23e67deb4a18d172c357a7cff14c}{set\+Range()}} and \mbox{\hyperlink{class_slider_a430a5c4e56b39dd622f5800f787e0822}{set\+Value()}} methods.

There are also lots of custom tweaks you can do by subclassing and overriding some of the virtual methods, such as changing the scaling, changing the format of the text display, custom ways of limiting the values, etc.

You can register \mbox{\hyperlink{class_slider_1_1_listener}{Slider\+::\+Listener}} objects with a slider, and they\textquotesingle{}ll be called when the value changes.

Called when a modifier key is pressed or released.

Whenever the shift, control, alt or command keys are pressed or released, this method will be called.

The component that is currently under the main mouse pointer will be tried first and, if there is no component currently under the pointer, the component that currently has the keyboard focus will have this method called. Remember that a component will only be given the focus if its \mbox{\hyperlink{class_component_a6a1f21a76b971d54870cb5c32c041055}{set\+Wants\+Keyboard\+Focus()}} method has been used to enable this.

The default implementation of this method actually calls its parent\textquotesingle{}s modifier\+Keys\+Changed method, so that focused components which aren\textquotesingle{}t interested in this will give their parents a chance to act on the event instead.

Sets up a skew factor to alter the way values are distributed.

You may want to use a range of values on the slider where more accuracy is required towards one end of the range, so this will logarithmically spread the values across the length of the slider.

If the factor is \texorpdfstring{$<$}{<} 1.\+0, the lower end of the range will fill more of the slider\textquotesingle{}s length; if the factor is \texorpdfstring{$>$}{>} 1.\+0, the upper end of the range will be expanded instead. A factor of 1.\+0 doesn\textquotesingle{}t skew it at all.

If symmetric\+Skew is true, the skew factor applies from the middle of the slider to each of its ends.

To set the skew position by using a mid-\/point, use the \mbox{\hyperlink{class_slider_a2d948c662c1f636810d3125d8b3f0d47}{set\+Skew\+Factor\+From\+Mid\+Point()}} method instead.
