\doxysubsection{Detailed Description}
A class for receiving events from a \mbox{\hyperlink{class_scroll_bar}{Scroll\+Bar}}.

You can register a \mbox{\hyperlink{class_scroll_bar_1_1_listener}{Scroll\+Bar\+::\+Listener}} with a \mbox{\hyperlink{class_scroll_bar}{Scroll\+Bar}} using the \mbox{\hyperlink{class_scroll_bar_a231620a2e529f6c1ff846d8b5faccc80}{Scroll\+Bar\+::add\+Listener()}} method, and it will be called when the bar\textquotesingle{}s position changes.
