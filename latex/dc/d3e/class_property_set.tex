\doxysubsection{Detailed Description}
A set of named property values, which can be strings, integers, floating point, etc.

Effectively, this just wraps a \mbox{\hyperlink{class_string_pair_array}{String\+Pair\+Array}} in an interface that makes it easier to load and save types other than strings.

See the \mbox{\hyperlink{class_properties_file}{Properties\+File}} class for a subclass of this, which automatically broadcasts change messages and saves/loads the list from a file.

Sets up a second Poperty\+Set that will be used to look up any values that aren\textquotesingle{}t set in this one.

If you set this up to be a pointer to a second property set, then whenever one of the \mbox{\hyperlink{class_property_set_afa005ac125096ad476021d347f56ff4c}{get\+Value()}} methods fails to find an entry in this set, it will look up that value in the fallback set, and if it finds it, it will return that.

Make sure that you don\textquotesingle{}t delete the fallback set while it\textquotesingle{}s still being used by another set! To remove the fallback set, just call this method with a null pointer.
