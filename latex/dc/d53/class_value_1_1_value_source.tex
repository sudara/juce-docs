\doxysubsection{Detailed Description}
Used internally by the \mbox{\hyperlink{class_value}{Value}} class as the base class for its shared value objects.

The \mbox{\hyperlink{class_value}{Value}} class is essentially a reference-\/counted pointer to a shared instance of a \mbox{\hyperlink{class_value_1_1_value_source}{Value\+Source}} object. If you\textquotesingle{}re feeling adventurous, you can create your own custom \mbox{\hyperlink{class_value_1_1_value_source}{Value\+Source}} classes to allow \mbox{\hyperlink{class_value}{Value}} objects to represent your own custom data items.
