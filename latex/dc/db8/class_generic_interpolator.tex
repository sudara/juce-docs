\subsubsection*{template$<$class Interpolator\+Traits, \mbox{\hyperlink{group__juce__audio__plugin__client-_unity_ga966b05a42d98c7296097e073c627dcd0}{int}} memory\+Size$>$\newline
class Generic\+Interpolator$<$ Interpolator\+Traits, memory\+Size $>$}
An interpolator base class for resampling streams of floats.

Note that the resamplers are stateful, so when there\textquotesingle{}s a break in the continuity of the input stream you\textquotesingle{}re feeding it, you should call \mbox{\hyperlink{class_generic_interpolator_a44d513d729c6dd222d496fcf73bc7818}{reset()}} before feeding it any new data. And like with any other stateful filter, if you\textquotesingle{}re resampling multiple channels, make sure each one uses its own interpolator object.
