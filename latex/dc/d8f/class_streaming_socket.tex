\doxysubsection{Detailed Description}
A wrapper for a streaming (TCP) socket.

This allows low-\/level use of sockets; for an easier-\/to-\/use messaging layer on top of sockets, you could also try the \mbox{\hyperlink{class_interprocess_connection}{Interprocess\+Connection}} class.
