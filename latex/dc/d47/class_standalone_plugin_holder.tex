\doxysubsection{Detailed Description}
An object that creates and plays a standalone instance of an \mbox{\hyperlink{class_audio_processor}{Audio\+Processor}}.

The object will create your processor using the same \mbox{\hyperlink{group__juce__audio__plugin__client-utility_ga62f69a86baff96d3ee5c287e100adc66}{create\+Plugin\+Filter()}} function that the other plugin wrappers use, and will run it through the computer\textquotesingle{}s audio/\+MIDI devices using \mbox{\hyperlink{class_audio_device_manager}{Audio\+Device\+Manager}} and \mbox{\hyperlink{class_audio_processor_player}{Audio\+Processor\+Player}}.

Creates an instance of the default plugin.

The settings object can be a \mbox{\hyperlink{class_property_set}{Property\+Set}} that the class should use to store its settings -\/ the take\+Ownership\+Of\+Settings indicates whether this object will delete the settings automatically when no longer needed. The settings can also be nullptr.

A default device name can be passed in.

Preferably a complete setup options object can be used, which takes precedence over the preferred\+Default\+Device\+Name and allows you to select the input \& output device names, sample rate, buffer size etc.
