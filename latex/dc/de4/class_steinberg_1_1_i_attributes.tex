\doxysubsection{Detailed Description}
Object Data Archive Interface.
\begin{DoxyItemize}
\item \mbox{[}host imp\mbox{]}
\item store data/objects/binary/subattributes in the archive
\item read stored data from the archive
\end{DoxyItemize}

All data stored to the archive are identified by a string (IAttr\+ID), which must be unique on each IAttribute level.

The basic set/get methods make use of the \mbox{\hyperlink{class_steinberg_1_1_f_variant}{FVariant}} class defined in \textquotesingle{}\mbox{\hyperlink{funknown_8h}{funknown.\+h}}\textquotesingle{}. For a more convenient usage of this interface, you should use the functions defined in namespace PAttributes (public.\+sdk/source/common/pattributes.h+cpp) !!
