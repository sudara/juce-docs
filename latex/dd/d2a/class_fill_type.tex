\doxysubsection{Detailed Description}
Represents a colour or fill pattern to use for rendering paths.

This is used by the \mbox{\hyperlink{class_graphics}{Graphics}} and \mbox{\hyperlink{class_drawable_path}{Drawable\+Path}} classes as a way to encapsulate a brush type. It can either be a solid colour, a gradient, or a tiled image.
