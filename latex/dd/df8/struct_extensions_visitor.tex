\doxysubsection{Detailed Description}
Create a derived implementation of this class and pass it to \mbox{\hyperlink{class_audio_plugin_instance_af0cc34ecbea95a99137c426cdf51714a}{Audio\+Plugin\+Instance\+::get\+Extensions()}} to retrieve format-\/specific information about a plugin instance.

Note that the references passed to the visit member functions are only guaranteed to live for the duration of the function call, so don\textquotesingle{}t store pointers to these objects! If you need to store and reuse format-\/specific information, it is recommended to copy the result of the function calls that you care about. For example, you should store the result of VST\+::get\+AEffect\+Ptr() rather than storing a pointer to the VST instance.
