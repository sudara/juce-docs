\doxysubsection{Detailed Description}
A type of timer class that can run multiple timers with different frequencies, all of which share a single callback.

This class is very similar to the \mbox{\hyperlink{class_timer}{Timer}} class, but allows you run multiple separate timers, where each one has a unique ID number. The methods in this class are exactly equivalent to those in \mbox{\hyperlink{class_timer}{Timer}}, but with the addition of this ID number.

To use it, you need to create a subclass of \mbox{\hyperlink{class_multi_timer}{Multi\+Timer}}, implementing the \mbox{\hyperlink{class_multi_timer_a16dde865d7aaf32ba2649f498518e7a7}{timer\+Callback()}} method. Then you can start timers with \mbox{\hyperlink{class_multi_timer_ae1a9966181d6680a15cebb95168093b5}{start\+Timer()}}, and each time the callback is triggered, it passes in the ID of the timer that caused it.
