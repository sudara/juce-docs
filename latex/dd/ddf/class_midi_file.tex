\doxysubsection{Detailed Description}
Reads/writes standard midi format files.

To read a midi file, create a \mbox{\hyperlink{class_midi_file}{Midi\+File}} object and call its \mbox{\hyperlink{class_midi_file_a4d59151ad6346b5f54cfe352d46478b4}{read\+From()}} method. You can then get the individual midi tracks from it using the \mbox{\hyperlink{class_midi_file_a3821fee8d3e8a49fbc129de0f00d6c89}{get\+Track()}} method.

To write a file, create a \mbox{\hyperlink{class_midi_file}{Midi\+File}} object, add some \mbox{\hyperlink{class_midi_message_sequence}{Midi\+Message\+Sequence}} objects to it using the \mbox{\hyperlink{class_midi_file_a9ff930286a6fa4531df163480b22a313}{add\+Track()}} method, and then call its \mbox{\hyperlink{class_midi_file_a7309d707566e3f943bcd29581c89c921}{write\+To()}} method to stream it out.

Returns the raw time format code that will be written to a stream.

After reading a midi file, this method will return the time-\/format that was read from the file\textquotesingle{}s header. It can be changed using the \mbox{\hyperlink{class_midi_file_ab18143cad013db21a71735b01aef2a9f}{set\+Ticks\+Per\+Quarter\+Note()}} or \mbox{\hyperlink{class_midi_file_ab63f442d05b8d7c2742de4515f6e8917}{set\+Smpte\+Time\+Format()}} methods.

It it\textquotesingle{}s negative, the upper byte indicates the frames-\/per-\/second (but negative), and the lower byte is the number of ticks per frame -\/ see \mbox{\hyperlink{class_midi_file_ab63f442d05b8d7c2742de4515f6e8917}{set\+Smpte\+Time\+Format()}}. \mbox{\Hypertarget{class_midi_file_a3821fee8d3e8a49fbc129de0f00d6c89}\label{class_midi_file_a3821fee8d3e8a49fbc129de0f00d6c89}}

Reads a midi file format stream.

After calling this, you can get the tracks that were read from the file by using the \mbox{\hyperlink{class_midi_file_a908f8db88d0740c8de225c31660d71a0}{get\+Num\+Tracks()}} and \mbox{\hyperlink{class_midi_file_a3821fee8d3e8a49fbc129de0f00d6c89}{get\+Track()}} methods.

The timestamps of the midi events in the tracks will represent their positions in terms of midi ticks. To convert them to seconds, use the \mbox{\hyperlink{class_midi_file_af66c3497ec6109a9c7c1cf1f474c8b95}{convert\+Timestamp\+Ticks\+To\+Seconds()}} method.
