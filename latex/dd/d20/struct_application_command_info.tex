\doxysubsection{Detailed Description}
Holds information describing an application command.

This object is used to pass information about a particular command, such as its name, description and other usage flags.

When an \mbox{\hyperlink{class_application_command_target}{Application\+Command\+Target}} is asked to provide information about the commands it can perform, this is the structure gets filled-\/in to describe each one.
