\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_m_d5}{MD5}} checksum class.

Create one of these with a block of source data or a stream, and it calculates the \mbox{\hyperlink{class_m_d5}{MD5}} checksum of that data.

You can then retrieve this checksum as a 16-\/byte block, or as a hex string. \begin{DoxySeeAlso}{See also}
