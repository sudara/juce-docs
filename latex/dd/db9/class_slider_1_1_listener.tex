\doxysubsection{Detailed Description}
A class for receiving callbacks from a \mbox{\hyperlink{class_slider}{Slider}}.

To be told when a slider\textquotesingle{}s value changes, you can register a \mbox{\hyperlink{class_slider_1_1_listener}{Slider\+::\+Listener}} object using \mbox{\hyperlink{class_slider_a261f4886ed27ab0f94b1cfb6e9f44011}{Slider\+::add\+Listener()}}.
