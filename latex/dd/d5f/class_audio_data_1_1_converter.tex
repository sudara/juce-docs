\doxysubsection{Detailed Description}
A base class for objects that are used to convert between two different sample formats.

The \mbox{\hyperlink{class_audio_data_1_1_converter_instance}{Audio\+Data\+::\+Converter\+Instance}} implements this base class and can be templated, so you can create an instance that converts between two particular formats, and then store this in the abstract base class.
