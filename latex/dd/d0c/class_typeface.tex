\doxysubsection{Detailed Description}
A typeface represents a size-\/independent font.

This base class is abstract, but calling \mbox{\hyperlink{class_typeface_a2ce51562cc51437352c4d47777185f35}{create\+System\+Typeface\+For()}} will return a platform-\/specific subclass that can be used.

The \mbox{\hyperlink{class_custom_typeface}{Custom\+Typeface}} subclass allow you to build your own typeface, and to load and save it in the JUCE typeface format.

Normally you should never need to deal directly with \mbox{\hyperlink{class_typeface}{Typeface}} objects -\/ the \mbox{\hyperlink{class_font}{Font}} class does everything you typically need for rendering text.
