\doxysubsection{Detailed Description}
\subsubsection*{template$<$class Object\+Type$>$\newline
class Optional\+Scoped\+Pointer$<$ Object\+Type $>$}
Holds a pointer to an object which can optionally be deleted when this pointer goes out of scope.

This acts in many ways like a std\+::unique\+\_\+ptr, but allows you to specify whether or not the object is deleted.
