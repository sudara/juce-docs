\doxysubsection{Detailed Description}
Receives change event callbacks that are sent out by a \mbox{\hyperlink{class_change_broadcaster}{Change\+Broadcaster}}.

A \mbox{\hyperlink{class_change_broadcaster}{Change\+Broadcaster}} keeps a set of listeners to which it broadcasts a message when the \mbox{\hyperlink{class_change_broadcaster_a10f01d1c0a7fde85355321985a682cbd}{Change\+Broadcaster\+::send\+Change\+Message()}} method is called. A subclass of \mbox{\hyperlink{class_change_listener}{Change\+Listener}} is used to receive these callbacks.

Note that the major difference between an \mbox{\hyperlink{class_action_listener}{Action\+Listener}} and a \mbox{\hyperlink{class_change_listener}{Change\+Listener}} is that for a \mbox{\hyperlink{class_change_listener}{Change\+Listener}}, multiple changes will be coalesced into fewer callbacks, but Action\+Listeners perform one callback for every event posted.
