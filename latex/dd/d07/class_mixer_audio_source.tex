\doxysubsection{Detailed Description}
An \mbox{\hyperlink{class_audio_source}{Audio\+Source}} that mixes together the output of a set of other Audio\+Sources.

Input sources can be added and removed while the mixer is running as long as their \mbox{\hyperlink{class_mixer_audio_source_af774982c4a8726436f3e505e459cdf18}{prepare\+To\+Play()}} and \mbox{\hyperlink{class_mixer_audio_source_aa4b64f534cadbd33e7aefcc6cee0f540}{release\+Resources()}} methods are called before and after adding them to the mixer.

Adds an input source to the mixer.

If the mixer is running you\textquotesingle{}ll need to make sure that the input source is ready to play by calling its \mbox{\hyperlink{class_mixer_audio_source_af774982c4a8726436f3e505e459cdf18}{prepare\+To\+Play()}} method before adding it.
