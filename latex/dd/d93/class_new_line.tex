\doxysubsection{Detailed Description}
This class is used for represent a new-\/line character sequence.

To write a new-\/line to a stream, you can use the predefined \textquotesingle{}new\+Line\textquotesingle{} variable, e.\+g. 
\begin{DoxyCode}{0}
\DoxyCodeLine{myOutputStream << \textcolor{stringliteral}{"{}Hello World"{}} << \mbox{\hyperlink{group__juce__core-text_gabd231b88341c3e2c328178d34f3ce8fb}{newLine}} << \mbox{\hyperlink{group__juce__core-text_gabd231b88341c3e2c328178d34f3ce8fb}{newLine}};}

\end{DoxyCode}


The exact character sequence that will be used for the new-\/line can be set and retrieved with \mbox{\hyperlink{class_output_stream_a3cd9133af8d3e6a04b8c983290649909}{Output\+Stream\+::set\+New\+Line\+String()}} and \mbox{\hyperlink{class_output_stream_a8d814d7fc873ab1799778b79d45f3951}{Output\+Stream\+::get\+New\+Line\+String()}}.
