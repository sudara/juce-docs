\doxysubsection{Detailed Description}
A base class for listeners that want to know about changes to an \mbox{\hyperlink{class_audio_processor_parameter}{Audio\+Processor\+Parameter}}.

Use \mbox{\hyperlink{class_audio_processor_parameter_a1e54fead318e3f817cde15bceae589ed}{Audio\+Processor\+Parameter\+::add\+Listener()}} to register your listener with an \mbox{\hyperlink{class_audio_processor_parameter}{Audio\+Processor\+Parameter}}.

This \mbox{\hyperlink{class_audio_processor_parameter_1_1_listener}{Listener}} replaces most of the functionality in the \mbox{\hyperlink{class_audio_processor_listener}{Audio\+Processor\+Listener}} class, which will be deprecated and removed.

Indicates that a parameter change gesture has started.

E.\+g. if the user is dragging a slider, this would be called with gesture\+Is\+Starting being true when they first press the mouse button, and it will be called again with gesture\+Is\+Starting being false when they release it.

IMPORTANT NOTE\+: This will be called synchronously, and many audio processors will call it during their audio callback. This means that not only has your handler code got to be completely thread-\/safe, but it\textquotesingle{}s also got to be VERY fast, and avoid blocking.
