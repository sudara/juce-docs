\doxysubsection{Detailed Description}
Class used to create a set of options to pass to the \mbox{\hyperlink{class_alert_window}{Alert\+Window}} and \mbox{\hyperlink{class_native_message_box}{Native\+Message\+Box}} methods for showing dialog boxes.

You can chain together a series of calls to this class\textquotesingle{}s methods to create a set of whatever options you want to specify.
