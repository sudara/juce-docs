\subsubsection*{template$<$class Object\+Type$>$\newline
class Linked\+List\+Pointer$<$ Object\+Type $>$\+::\+Appender}
Allows efficient repeated insertions into a list.

You can create an \mbox{\hyperlink{class_linked_list_pointer_1_1_appender}{Appender}} object which points to the last element in your list, and then repeatedly call \mbox{\hyperlink{class_linked_list_pointer_1_1_appender_a9cdd44669fe1fb53b54e0dfc30833c2b}{Appender\+::append()}} to add items to the end of the list in O(1) time.
