Returns a string to uniquely identify this item.

If you\textquotesingle{}re planning on using the \mbox{\hyperlink{class_tree_view_a7d7c90a3d4e7e1e76bef68b7d0787302}{Tree\+View\+::get\+Openness\+State()}} method, then these strings will be used to identify which nodes are open. The string should be unique amongst the item\textquotesingle{}s sibling items, but it\textquotesingle{}s ok for there to be duplicates at other levels of the tree.

If you\textquotesingle{}re not going to store the state, then it\textquotesingle{}s ok not to bother implementing this method.

Tells the tree whether this item can potentially be opened.

If your item could contain sub-\/items, this should return true; if it returns false then the tree will not try to open the item. This determines whether or not the item will be drawn with a \textquotesingle{}plus\textquotesingle{} button next to it.
