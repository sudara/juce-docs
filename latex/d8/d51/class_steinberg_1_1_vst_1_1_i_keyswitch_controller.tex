\doxysubsection{Detailed Description}
Extended plug-\/in interface \mbox{\hyperlink{class_steinberg_1_1_vst_1_1_i_edit_controller}{IEdit\+Controller}} for key switches support\+: \mbox{\hyperlink{class_steinberg_1_1_vst_1_1_i_keyswitch_controller}{Vst\+::\+IKeyswitch\+Controller}}

\item \mbox{[}released\+: 3.\+5.\+0\mbox{]}
\item \mbox{[}optional\mbox{]}

When a (instrument) plug-\/in supports such interface, the host could get from the plug-\/in the current set of used key switches (megatrig/articulation) for a given channel of a event bus and then automatically use them (like in Cubase 6) to create VST \mbox{\hyperlink{class_expression}{Expression}} Map (allowing to associated symbol to a given articulation / key switch).
