If the file doesn\textquotesingle{}t exist, it will first be created. If the file can\textquotesingle{}t be created or opened (for example, because the parent directory of the file does not exist), the \mbox{\hyperlink{class_file_output_stream_a616887c2791a85e35f00b6e38436d4be}{failed\+To\+Open()}} method will return true.

If the file already exists when opened, the stream\textquotesingle{}s write-\/position will be set to the end of the file. To overwrite an existing file, you can truncate it like this\+:
