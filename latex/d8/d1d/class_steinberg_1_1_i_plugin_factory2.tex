From the host\textquotesingle{}s point of view a plug-\/in module is a factory which can create a certain kind of object(s). The interface \mbox{\hyperlink{class_steinberg_1_1_i_plugin_factory}{IPlugin\+Factory}} provides methods to get information about the classes exported by the plug-\/in and a mechanism to create instances of these classes (that usually define the \mbox{\hyperlink{class_steinberg_1_1_i_plugin_base}{IPlugin\+Base}} interface).
