\doxysubsection{Detailed Description}
A graphical effect filter that can be applied to components.

An \mbox{\hyperlink{class_image_effect_filter}{Image\+Effect\+Filter}} can be applied to the image that a component paints before it hits the screen.

This is used for adding effects like shadows, blurs, etc.
