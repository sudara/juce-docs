If you want to set the value of a parameter internally, e.\+g. from your editor component, then don\textquotesingle{}t call this directly -\/ instead, use the \mbox{\hyperlink{class_audio_processor_parameter_ac9b67f35339db50d2bd9a026d89390e1}{set\+Value\+Notifying\+Host()}} method, which will also send a message to the host telling it about the change.
