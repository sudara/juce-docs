Indicates that the plug-\/in could or not support Prefetch (dynamically)\+: \mbox{\hyperlink{class_steinberg_1_1_vst_1_1_i_prefetchable_support}{Vst\+::\+IPrefetchable\+Support}}

The plug-\/in should implement this interface if it needs to dynamically change between prefetchable or not. By default (without implementing this interface) the host decides in which mode the plug-\/in is processed.
