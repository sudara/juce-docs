\doxysubsection{Detailed Description}
A utility class for fitting a set of objects whose sizes can vary between a minimum and maximum size, into a space.

This is a trickier algorithm than it would first seem, so I\textquotesingle{}ve put it in this class to allow it to be shared by various bits of code.

To use it, create one of these objects, call \mbox{\hyperlink{class_stretchable_object_resizer_a2f4ade92ab81712e4556770ecfb81cd0}{add\+Item()}} to add the list of items you need, then call \mbox{\hyperlink{class_stretchable_object_resizer_ad91dce94a524b769b10c663eec6f0b4e}{resize\+To\+Fit()}}, which will change all their sizes. You can then retrieve the new sizes with \mbox{\hyperlink{class_stretchable_object_resizer_aca2902b220e5ec11d050e013f2de4de9}{get\+Item\+Size()}} and \mbox{\hyperlink{class_stretchable_object_resizer_a6ae31268a50a1c715e39b28bfd82f560}{get\+Num\+Items()}}.

It\textquotesingle{}s currently used by the \mbox{\hyperlink{class_table_header_component}{Table\+Header\+Component}} for stretching out the table headings to fill the table\textquotesingle{}s width.

Resizes all the items to fit this amount of space.

This will attempt to fit them in without exceeding each item\textquotesingle{}s minimum and maximum sizes. In cases where none of the items can be expanded or enlarged any further, the final size may be greater or less than the size passed in.

After calling this method, you can retrieve the new sizes with the \mbox{\hyperlink{class_stretchable_object_resizer_aca2902b220e5ec11d050e013f2de4de9}{get\+Item\+Size()}} method.
