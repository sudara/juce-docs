\doxysubsection{Detailed Description}
Represents an open\+GL uniform value. After a program has been linked, you can create \mbox{\hyperlink{struct_open_g_l_shader_program_1_1_uniform}{Uniform}} objects to let you set the uniforms that your shaders use.

Be careful not to call the \mbox{\hyperlink{struct_open_g_l_shader_program_1_1_uniform_a551ac717f9a07475a90a946fcee4b678}{set()}} functions unless the appropriate program is loaded into the current context.
