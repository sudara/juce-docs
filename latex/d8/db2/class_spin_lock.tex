\doxysubsection{Detailed Description}
A simple spin-\/lock class that can be used as a simple, low-\/overhead mutex for uncontended situations.

Note that unlike a \mbox{\hyperlink{class_critical_section}{Critical\+Section}}, this type of lock is not re-\/entrant, and may be less efficient when used in a highly contended situation, but it\textquotesingle{}s very small and requires almost no initialisation. It\textquotesingle{}s most appropriate for simple situations where you\textquotesingle{}re only going to hold the lock for a very brief time.

Acquires the lock. This will block until the lock has been successfully acquired by this thread. Note that a \mbox{\hyperlink{class_spin_lock}{Spin\+Lock}} is NOT re-\/entrant, and is not smart enough to know whether the caller thread already has the lock -\/ so if a thread tries to acquire a lock that it already holds, this method will never return!
