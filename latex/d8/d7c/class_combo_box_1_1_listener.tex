\doxysubsection{Detailed Description}
A class for receiving events from a \mbox{\hyperlink{class_combo_box}{Combo\+Box}}.

You can register a \mbox{\hyperlink{class_combo_box_1_1_listener}{Combo\+Box\+::\+Listener}} with a \mbox{\hyperlink{class_combo_box}{Combo\+Box}} using the \mbox{\hyperlink{class_combo_box_a750af2fdf946a248d661fa4d2ca72e37}{Combo\+Box\+::add\+Listener()}} method, and it will be called when the selected item in the box changes.
