\doxysubsection{Detailed Description}
A timer for measuring performance of code and dumping the results to a file.

e.\+g. 
\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{class_performance_counter}{PerformanceCounter}} pc (\textcolor{stringliteral}{"{}fish"{}}, 50, \textcolor{stringliteral}{"{}/temp/myfishlog.txt"{}});}
\DoxyCodeLine{}
\DoxyCodeLine{\textcolor{keywordflow}{for} (;;)}
\DoxyCodeLine{\{}
\DoxyCodeLine{    pc.start();}
\DoxyCodeLine{}
\DoxyCodeLine{    doSomethingFishy();}
\DoxyCodeLine{}
\DoxyCodeLine{    pc.stop();}
\DoxyCodeLine{\}}

\end{DoxyCode}


In this example, the time of each period between calling start/stop will be measured and averaged over 50 runs, and the results printed to a file every 50 times round the loop.
