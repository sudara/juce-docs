\doxysubsection{Detailed Description}
An arbitrarily large integer class.

A \mbox{\hyperlink{class_big_integer}{Big\+Integer}} can be used in a similar way to a normal integer, but has no size limit (except for memory and performance constraints).

Negative values are possible, but the value isn\textquotesingle{}t stored as 2s-\/complement, so be careful if you use negative values and look at the values of individual bits.
