\doxysubsection{Detailed Description}
Base class for traversing components.

If you need custom focus or keyboard focus traversal for a component you can create a subclass of \mbox{\hyperlink{class_component_traverser}{Component\+Traverser}} and return it from \mbox{\hyperlink{class_component_a138dc2e3c7f4836d7c127409dab8a56d}{Component\+::create\+Focus\+Traverser()}} or \mbox{\hyperlink{class_component_a8dc872ef83fedecd89cf814d28236cd0}{Component\+::create\+Keyboard\+Focus\+Traverser()}}.
