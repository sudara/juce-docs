\doxysubsection{Detailed Description}
A type of \mbox{\hyperlink{class_audio_source}{Audio\+Source}} which can be repositioned.

The basic \mbox{\hyperlink{class_audio_source}{Audio\+Source}} just streams continuously with no idea of a current time or length, so the \mbox{\hyperlink{class_positionable_audio_source}{Positionable\+Audio\+Source}} is used for a finite stream that has a current read position.

Tells the stream to move to a new position.
