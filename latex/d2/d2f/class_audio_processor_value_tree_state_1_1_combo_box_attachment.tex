\doxysubsection{Detailed Description}
An object of this class maintains a connection between a \mbox{\hyperlink{class_combo_box}{Combo\+Box}} and a parameter in an \mbox{\hyperlink{class_audio_processor_value_tree_state}{Audio\+Processor\+Value\+Tree\+State}}.

Combobox items will be spaced linearly across the range of the parameter. For example if the range is specified by Normalisable\+Range$<$float$>$ (-\/0.\+5f, 0.\+5f, 0.\+5f) and you add three items then the first will be mapped to a value of -\/0.\+5, the second to 0, and the third to 0.\+5.

During the lifetime of this \mbox{\hyperlink{class_audio_processor_value_tree_state_1_1_combo_box_attachment}{Combo\+Box\+Attachment}} object, it keeps the two things in sync, making it easy to connect a combo box to a parameter. When this object is deleted, the connection is broken. Make sure that your \mbox{\hyperlink{class_audio_processor_value_tree_state}{Audio\+Processor\+Value\+Tree\+State}} and \mbox{\hyperlink{class_combo_box}{Combo\+Box}} aren\textquotesingle{}t deleted before this object!
