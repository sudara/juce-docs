\doxysubsection{Detailed Description}
An object which runs a thread to repeatedly broadcast the existence of a discoverable service.

To use, simply create an instance of an \mbox{\hyperlink{struct_network_service_discovery_1_1_advertiser}{Advertiser}} and it\textquotesingle{}ll broadcast until you delete it.
