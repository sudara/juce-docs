\doxysubsection{Detailed Description}
Expresses a coordinate as a dynamically evaluated expression.

When using relative coordinates to position components, the following symbols are available\+:
\begin{DoxyItemize}
\item \char`\"{}left\char`\"{}, \char`\"{}right\char`\"{}, \char`\"{}top\char`\"{}, \char`\"{}bottom\char`\"{} refer to the position of those edges in this component, so e.\+g. for a component whose width is always 100, you might set the right edge to the \char`\"{}left + 100\char`\"{}.
\item \char`\"{}\mbox{[}id\mbox{]}.\+left\char`\"{}, \char`\"{}\mbox{[}id\mbox{]}.\+right\char`\"{}, \char`\"{}\mbox{[}id\mbox{]}.\+top\char`\"{}, \char`\"{}\mbox{[}id\mbox{]}.\+bottom\char`\"{}, \char`\"{}\mbox{[}id\mbox{]}.\+width\char`\"{}, \char`\"{}\mbox{[}id\mbox{]}.\+height\char`\"{}, where \mbox{[}id\mbox{]} is the identifier of one of this component\textquotesingle{}s siblings. A component\textquotesingle{}s identifier is set with \mbox{\hyperlink{class_component_ac41c215031a087064add6d3a180abd1d}{Component\+::set\+Component\+ID()}}. So for example if you want your component to always be 50 pixels to the right of the one called \char`\"{}xyz\char`\"{}, you could set your left edge to be \char`\"{}xyz.\+right + 50\char`\"{}.
\item Instead of an \mbox{[}id\mbox{]}, you can use the name \char`\"{}parent\char`\"{} to refer to this component\textquotesingle{}s parent. Like any other component, these values are relative to their component\textquotesingle{}s parent, so \char`\"{}parent.\+right\char`\"{} won\textquotesingle{}t be very useful for positioning a component because it refers to a position with the parent\textquotesingle{}s parent.. but \char`\"{}parent.\+width\char`\"{} can be used for setting positions relative to the parent\textquotesingle{}s size. E.\+g. to make a 10x10 component which remains 1 pixel away from its parent\textquotesingle{}s bottom-\/right, you could use \char`\"{}right -\/ 10, bottom -\/ 10, parent.\+width -\/ 1, parent.\+height -\/ 1\char`\"{}.
\item The name of one of the parent component\textquotesingle{}s markers can also be used as a symbol. For markers to be used, the parent component must implement its Component\+::get\+Markers() method, and return at least one valid \mbox{\hyperlink{class_marker_list}{Marker\+List}}. So if you want your component\textquotesingle{}s top edge to be 10 pixels below the marker called \char`\"{}foobar\char`\"{}, you\textquotesingle{}d set it to \char`\"{}foobar + 10\char`\"{}.
\end{DoxyItemize}

See the \mbox{\hyperlink{class_expression}{Expression}} class for details about the operators that are supported, but for example if you wanted to make your component remains centred within its parent with a size of 100, 100, you could express it as\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{myComp.setBounds (RelativeBounds (\textcolor{stringliteral}{"{}parent.width / 2 -\/ 50, parent.height / 2 -\/ 50, left + 100, top + 100"{}}));}

\end{DoxyCode}
 ..or an alternative way to achieve the same thing\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{myComp.setBounds (RelativeBounds (\textcolor{stringliteral}{"{}right -\/ 100, bottom -\/ 100, parent.width / 2 + 50, parent.height / 2 + 50"{}}));}

\end{DoxyCode}


Or if you wanted a 100x100 component whose top edge is lined up to a marker called \char`\"{}top\+Marker\char`\"{} and which is positioned 50 pixels to the right of another component called \char`\"{}other\+Comp\char`\"{}, you could write\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{myComp.setBounds (RelativeBounds (\textcolor{stringliteral}{"{}otherComp.right + 50, topMarker, left + 100, top + 100"{}}));}

\end{DoxyCode}


Be careful not to make your coordinate expressions recursive, though, or exceptions and assertions will be thrown!
