Registers a callback that will be called when a file descriptor is ready for I/O.

This will add the given file descriptor to the internal set of file descriptors that will be passed to the poll() call. When this file descriptor has data to read the read\+Callback will be called.


\begin{DoxyParams}{Parameters}
{\em fd} & the file descriptor to be monitored \\
\hline
{\em read\+Callback} & a callback that will be called when the file descriptor has data to read. The file descriptor will be passed as an argument \\
