\doxysubsection{Detailed Description}
A path object that consists of \mbox{\hyperlink{class_relative_point}{Relative\+Point}} coordinates rather than the normal fixed ones.

One of these paths can be converted into a \mbox{\hyperlink{class_path}{Path}} object for drawing and manipulation, but unlike a \mbox{\hyperlink{class_path}{Path}}, its points can be dynamic instead of just fixed.
