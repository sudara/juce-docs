\doxysubsection{Detailed Description}
Represents the range of this value, if supported.

Return one of these from the {\ttfamily \mbox{\hyperlink{class_accessibility_value_interface_a0be6d4ed858468222f6110c230ece502}{get\+Range()}}} method, providing a minimum, maximum, and interval value for the range to indicate that this is a ranged value.

The default state is an \char`\"{}invalid\char`\"{} range, indicating that the accessibility element does not support ranged values.
