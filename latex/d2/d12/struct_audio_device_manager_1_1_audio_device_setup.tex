\doxysubsection{Detailed Description}
This structure holds a set of properties describing the current audio setup.

An \mbox{\hyperlink{class_audio_device_manager}{Audio\+Device\+Manager}} uses this class to save/load its current settings, and to specify your preferred options when opening a device.
