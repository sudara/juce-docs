\doxysubsection{Detailed Description}
Adds a focus outline to a component.

This object creates and manages a component that sits on top of a target component. It will track the position of the target component and will be brought to the front with the tracked component.

Use the \mbox{\hyperlink{class_component_a75d4febc425c0c0ffe4567e265b45ef2}{Component\+::set\+Has\+Focus\+Outline()}} method to indicate that a component should have a focus outline drawn around it, and when it receives keyboard focus one of these objects will be created using the \mbox{\hyperlink{class_look_and_feel_a39f2061a27e007e214a607dbf0e31baf}{Look\+And\+Feel\+::create\+Focus\+Outline\+For\+Component()}} method. You can override this method in your own \mbox{\hyperlink{class_look_and_feel}{Look\+And\+Feel}} classes to draw a custom outline if required.
