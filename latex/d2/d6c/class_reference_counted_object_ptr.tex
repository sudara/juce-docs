\subsubsection*{template$<$class Object\+Type$>$\newline
class Reference\+Counted\+Object\+Ptr$<$ Object\+Type $>$}
A smart-\/pointer class which points to a reference-\/counted object.

The template parameter specifies the class of the object you want to point to -\/ the easiest way to make a class reference-\/countable is to simply make it inherit from \mbox{\hyperlink{class_reference_counted_object}{Reference\+Counted\+Object}} or \mbox{\hyperlink{class_single_threaded_reference_counted_object}{Single\+Threaded\+Reference\+Counted\+Object}}, but if you need to, you can roll your own reference-\/countable class by implementing a set of methods called inc\+Reference\+Count(), dec\+Reference\+Count(), and dec\+Reference\+Count\+Without\+Deleting(). See \mbox{\hyperlink{class_reference_counted_object}{Reference\+Counted\+Object}} for examples of how these methods should behave.

When using this class, you\textquotesingle{}ll probably want to create a typedef to abbreviate the full templated name -\/ e.\+g.

\DoxyCodeLine{\{}

\DoxyCodeLine{    ...}
