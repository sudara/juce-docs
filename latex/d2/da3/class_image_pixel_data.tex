\doxysubsection{Detailed Description}
This is a base class for holding image data in implementation-\/specific ways.

You may never need to use this class directly -\/ it\textquotesingle{}s used internally by the \mbox{\hyperlink{class_image}{Image}} class to store the actual image data. To access pixel data directly, you should use \mbox{\hyperlink{class_image_1_1_bitmap_data}{Image\+::\+Bitmap\+Data}} rather than this class.

\mbox{\hyperlink{class_image_pixel_data}{Image\+Pixel\+Data}} objects are created indirectly, by subclasses of \mbox{\hyperlink{class_image_type}{Image\+Type}}. \begin{DoxySeeAlso}{See also}
\mbox{\hyperlink{class_image}{Image}}, \mbox{\hyperlink{class_image_type}{Image\+Type}}
\end{DoxySeeAlso}
