\doxysubsection{Detailed Description}
A universally unique 128-\/bit identifier.

This class generates very random unique numbers. It\textquotesingle{}s vanishingly unlikely that two identical UUIDs would ever be created by chance. The values are formatted to meet the RFC 4122 version 4 standard.

The class includes methods for saving the ID as a string or as raw binary data.
