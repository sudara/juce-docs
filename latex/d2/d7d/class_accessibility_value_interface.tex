\doxysubsection{Detailed Description}
An abstract interface representing the value of an accessibility element.

Values should be used when information needs to be conveyed which cannot be represented by the accessibility element\textquotesingle{}s label alone. For example, a gain slider with the label \char`\"{}\+Gain\char`\"{} needs to also provide a value for its position whereas a \char`\"{}\+Save\char`\"{} button does not.

This class allows for full control over the value text/numeric conversion, ranged, and read-\/only properties but in most cases you\textquotesingle{}ll want to use one of the derived classes below which handle some of this for you.
