Turns the slider\textquotesingle{}s current value into a text string.

Subclasses can override this to customise the formatting of the text-\/entry box.

The default implementation just turns the value into a string, using a number of decimal places based on the range interval. If a suffix string has been set using \mbox{\hyperlink{class_slider_ac416a101b5d9a504f61e2f50dc593f61}{set\+Text\+Value\+Suffix()}}, this will be appended to the text.
